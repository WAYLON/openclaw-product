# 股票 Agent 示例问法

## 标准问法
- 请先帮我说明你能处理什么，不能处理什么。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请帮我分析 600519 最近 3 个月的走势，按行情、基本面、新闻和风险四部分输出。
- 请帮我做一份 A 股教育板块的观察清单，并说明哪些指标值得继续跟踪。
- 请把这份回测结果整理成讲师能直接拿去讲的中文复盘摘要。

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
