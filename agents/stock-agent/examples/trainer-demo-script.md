# 股票 Agent 讲师演示脚本

## 一、开场介绍
- 先说明 股票 Agent 是平行独立 Agent，不依赖业务总控 Agent。
- 再说明它解决的核心问题：A 股研究辅助、风险提示、行情观察与研究摘要交付。

## 二、初始化演示
- 先故意触发一次需要 Key 的能力。
- 让客户看到中文初始化提示。
- 再演示如何配置凭证并继续。

## 三、标准演示问题
- 请分析 600519 最近 3 个月走势，并按结论、依据、风险输出。
- 请生成一份教育板块观察清单。
- 请把这份回测结果改写成讲师可讲的复盘摘要。

## 四、讲解顺序建议
1. 先讲研究辅助定位
2. 再演示一只股票的完整分析链路
3. 最后强调风险边界与不能做的事

## 五、风险边界提醒
- 首发不做自动交易
- 不输出保证收益表达
- 不能替代正式投顾结论
