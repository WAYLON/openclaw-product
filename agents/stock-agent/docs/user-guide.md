# 股票 Agent 内部用户指南

## 核心定位
股票 Agent 主要面向交付后的终端用户，但这份内部版是给实施同学、开发同学、培训老师做实施和交付校验用的。

## 用户真正会用到的能力
- 代码解析
- 行情/K线
- 基本面/技术面
- 新闻情绪影响
- 研究摘要与风险提示

## 用户成功标准
- 知道这个 Agent 能做什么。
- 知道第一次怎么配置。
- 知道如何正确提问。
- 知道风险边界在哪里。

## 初始化关注点
- tradingagents_api_key（按部署方式）
- 行情源接入凭证（按部署方式）

## 内部交付提醒
- 不做自动交易
- 不做收益承诺
- 不替代投顾
