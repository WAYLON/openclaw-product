# 股票 Agent 内部讲师指南

## 讲课目的
帮助实施老师快速理解这个 Agent 的讲法、演示顺序和风险边界，而不是临时拼凑话术。

## 建议授课流程
1. 先讲这个 Agent 解决什么问题。
2. 再讲它为什么必须独立安装、独立绑定渠道。
3. 再演示首次初始化。
4. 再跑一轮标准问题。
5. 最后讲风险边界和不适用场景。

## 建议演示问题
- 请分析 600519 最近 3 个月走势，并按结论、依据、风险输出。
- 请生成一份板块观察清单。
- 请把回测结果改成讲师可讲的复盘摘要。

## 讲师注意事项
- 不做自动交易
- 不做收益承诺
- 不替代投顾
