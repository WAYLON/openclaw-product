# 像素修图师 示例问法

## 标准问法
- 请先帮我说明你能处理什么，不能处理什么。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请先帮我预检这张老照片适合做哪些修复，再给出 CPU 和 GPU 两种方案。
- 请把这批商品图做背景移除，并告诉我哪些图还需要人工复查。
- 请帮我把这张模糊人像做增强，但先告诉我能恢复到什么程度，不能保证什么。

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
