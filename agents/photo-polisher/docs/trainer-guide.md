# 像素修图师 内部讲师指南

## 讲课目的
帮助实施老师快速理解这个 Agent 的讲法、演示顺序和风险边界，而不是临时拼凑话术。

## 建议授课流程
1. 先讲这个 Agent 解决什么问题。
2. 再讲它为什么必须独立安装、独立绑定渠道。
3. 再演示首次初始化。
4. 再跑一轮标准问题。
5. 最后讲风险边界和不适用场景。

## 建议演示问题
- 请先预检这张老照片适合哪些修复。
- 请把这批商品图做背景移除并标记人工复查项。
- 请说明 CPU 和 GPU 两种方案区别。

## 讲师注意事项
- 不承诺无限恢复
- 不伪造证据图片
- 不把猜测补全当真实修复
