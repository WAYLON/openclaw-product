# 助贷 Agent 内部用户指南

## 核心定位
助贷 Agent 主要面向交付后的终端用户，但这份内部版是给实施同学、开发同学、培训老师做实施和交付校验用的。

## 用户真正会用到的能力
- 线索采集
- 资格预检
- 材料清单
- OCR解析
- 预约与合同流转说明

## 用户成功标准
- 知道这个 Agent 能做什么。
- 知道第一次怎么配置。
- 知道如何正确提问。
- 知道风险边界在哪里。

## 初始化关注点
- formbricks_api_key
- espocrm_api_key
- calcom_api_key
- documenso_api_key
- ocr_service_key（按需）

## 内部交付提醒
- 不替代授信审批
- 不承诺放款结果
- 不建议规避合规
