# 助贷 Agent 内部示例问法

## 推荐联调问法
- 请围绕“留资初筛、材料清单、OCR 解析、合规提示”给我一个最适合现场演示的三步流程。
- 请把当前输出改成“客户能听懂、老师能讲”的版本。
- 请先列结论，再列依据，最后补风险边界和下一步建议。

## 推荐反问口径
- 如果资料不足，请先明确告诉用户还缺哪些输入。
- 如果涉及 key、外部系统或服务未初始化，请先返回中文初始化提示。
- 如果超出当前 Agent 能力边界，请明确说明应切换到哪个专业 Agent。

## 常见错误用法
- 什么上下文都不给，直接要求一次性全自动落地。
- 未初始化外部服务就要求直接读写第三方系统。
- 让当前 Agent 越界替其他专业 Agent 做最终业务判断。
