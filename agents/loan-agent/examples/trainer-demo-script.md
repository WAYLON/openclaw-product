# 助贷 Agent 讲师演示脚本

## 一、开场介绍
- 先说明 助贷 Agent 是平行独立 Agent，不依赖业务总控 Agent。
- 再说明它解决的核心问题：线索采集、资格预检、材料引导、预约与合规流程交付。

## 二、初始化演示
- 先故意触发一次需要 Key 的能力。
- 让客户看到中文初始化提示。
- 再演示如何配置凭证并继续。

## 三、标准演示问题
- 请先做资格预检并列出还缺哪些材料。
- 请把这份收入证明解析成中文摘要。
- 请生成一份面向客户的合规材料清单说明。

## 四、讲解顺序建议
1. 先讲流程定位
2. 再演示材料清单和预检
3. 最后讲合规边界

## 五、风险边界提醒
- 不替代授信审批
- 不承诺放款结果
- 不建议规避合规要求
