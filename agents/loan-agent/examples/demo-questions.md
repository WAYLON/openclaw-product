# 助贷 Agent 示例问法

## 标准问法
- 请先帮我说明你能处理什么，不能处理什么。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请根据这位客户的基本情况，先做助贷资格预检，并列出还缺哪些材料。
- 请把这份收入证明和银行流水解析成顾问能直接跟进的中文摘要。
- 请帮我生成一份面向客户的材料清单说明，要求口径稳妥、合规、不夸大。

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
