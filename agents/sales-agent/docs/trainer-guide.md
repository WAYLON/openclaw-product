# 客服 / 销售转化 Agent 内部讲师指南

## 讲课目的
帮助实施老师快速理解这个 Agent 的讲法、演示顺序和风险边界，而不是临时拼凑话术。

## 建议授课流程
1. 先讲这个 Agent 解决什么问题。
2. 再讲它为什么必须独立安装、独立绑定渠道。
3. 再演示首次初始化。
4. 再跑一轮标准问题。
5. 最后讲风险边界和不适用场景。

## 建议演示问题
- 请判断这段咨询属于FAQ、销售机会还是投诉。
- 请给一版客户回复和一版内部CRM建议。
- 请判断客户所处销售阶段并给出下一步动作。

## 讲师注意事项
- 不夸大承诺
- 不违规外呼
- 不把投诉场景硬推成交
