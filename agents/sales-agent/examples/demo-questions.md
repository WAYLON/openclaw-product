# 客服 / 销售转化 Agent 示例问法

## 标准问法
- 请先帮我说明你能处理什么，不能处理什么。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请把这段客户咨询判断成 FAQ、线索还是投诉，并给出推荐回复。
- 请帮我写一版适合客服继续跟进的中文回复，并同步给出 CRM 记录建议。
- 请根据这位客户的交流内容，判断他目前处于哪个销售阶段，下一步该推进什么。

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
