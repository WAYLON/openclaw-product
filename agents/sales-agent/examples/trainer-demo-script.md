# 客服 / 销售转化 Agent 讲师演示脚本

## 一、开场介绍
- 先说明 客服 / 销售转化 Agent 是平行独立 Agent，不依赖业务总控 Agent。
- 再说明它解决的核心问题：FAQ、线索识别、跟进推进、预约转化与记录沉淀。

## 二、初始化演示
- 先故意触发一次需要 Key 的能力。
- 让客户看到中文初始化提示。
- 再演示如何配置凭证并继续。

## 三、标准演示问题
- 请判断这段咨询属于 FAQ、销售机会还是投诉。
- 请给一版客户回复和一版内部 CRM 跟进建议。
- 请判断客户所处销售阶段并给出下一步动作。

## 四、讲解顺序建议
1. 先讲服务和销售一体化
2. 再演示回复与跟进
3. 最后讲投诉边界

## 五、风险边界提醒
- 不能夸大承诺
- 不能违规外呼
- 不能把投诉场景硬推成交
