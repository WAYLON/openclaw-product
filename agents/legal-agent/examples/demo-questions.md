# 法务 / 合同流程 Agent 示例问法

## 标准问法
- 请先帮我说明你能处理什么，不能处理什么。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请帮我解析这份合同，先提取关键条款，再标出需要重点人工复核的位置。
- 请把两个版本的合同做差异比较，并按付款、违约、交付三类列出变化。
- 请帮我整理一份适合交付经理讲解的签署流程说明，要求中文、流程化、可截图演示。

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
