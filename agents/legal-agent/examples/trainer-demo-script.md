# 法务 / 合同流程 Agent 讲师演示脚本

## 一、开场介绍
- 先说明 法务 / 合同流程 Agent 是平行独立 Agent，不依赖业务总控 Agent。
- 再说明它解决的核心问题：合同解析、条款比对、审批流转、签署跟踪与证据打包。

## 二、初始化演示
- 先故意触发一次需要 Key 的能力。
- 让客户看到中文初始化提示。
- 再演示如何配置凭证并继续。

## 三、标准演示问题
- 请解析这份合同并标出重点复核条款。
- 请比对两个版本合同。
- 请生成一份签署流程说明。

## 四、讲解顺序建议
1. 先讲合同流程而不是法律替代
2. 再演示解析和比对
3. 最后讲签署状态与留痕
