# 法务 / 合同流程 Agent 内部用户指南

## 核心定位
法务 / 合同流程 Agent 主要面向交付后的终端用户，但这份内部版是给实施同学、开发同学、培训老师做实施和交付校验用的。

## 用户真正会用到的能力
- 合同上传解析
- OCR与表格提取
- 条款风险提示
- 版本比对
- 签署状态跟踪

## 用户成功标准
- 知道这个 Agent 能做什么。
- 知道第一次怎么配置。
- 知道如何正确提问。
- 知道风险边界在哪里。

- 不自动签署文件
