# 教育 Agent 讲师演示脚本

## 一、开场介绍
- 先说明 教育 Agent 是平行独立 Agent，不依赖业务总控 Agent。
- 再说明它解决的核心问题：备课、题目、讲义解析、语音课堂接入与老师讲解交付。

## 二、初始化演示
- 先故意触发一次需要 Key 的能力。
- 让客户看到中文初始化提示。
- 再演示如何配置凭证并继续。

## 三、标准演示问题
- 请设计一节函数图像课的大纲。
- 请把这段课堂录音整理成中文纪要。
- 请把这份讲义改成老师可播报的语音串讲稿。

## 四、讲解顺序建议
1. 先讲教学设计能力
2. 再演示课堂资料解析
3. 最后演示语音转写或播报

## 五、风险边界提醒
- 不替代正式评分
- 不把语音误识别内容当事实
- 不替代老师做高风险学生判断
