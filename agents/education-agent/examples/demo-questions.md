# 教育 Agent 示例问法

## 标准问法
- 请先帮我说明你能处理什么，不能处理什么。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请把这段课堂录音整理成中文课堂纪要，并列出可继续追问的知识点。
- 请把这份教学提纲改成适合老师现场播报的语音串讲稿。
- 请根据学生的一段口语作答，给出中文点评和下一轮练习建议。

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
