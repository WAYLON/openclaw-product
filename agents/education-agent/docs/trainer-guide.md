# 教育 Agent 内部讲师指南

## 讲课目的
帮助实施老师快速理解这个 Agent 的讲法、演示顺序和风险边界，而不是临时拼凑话术。

## 建议授课流程
1. 先讲这个 Agent 解决什么问题。
2. 再讲它为什么必须独立安装、独立绑定渠道。
3. 再演示首次初始化。
4. 再跑一轮标准问题。
5. 最后讲风险边界和不适用场景。

## 建议演示问题
- 请设计一节函数图像课的大纲。
- 请把这段课堂录音整理成中文纪要。
- 请把讲义改成老师可播报的语音串讲稿。

## 讲师注意事项
- 不替代正式评分
- 不把语音误识别当事实
- 不替代老师做高风险判断
