# 教育 Agent 内部用户指南

## 核心定位
教育 Agent 主要面向交付后的终端用户，但这份内部版是给实施同学、开发同学、培训老师做实施和交付校验用的。

## 用户真正会用到的能力
- 课程主题规划
- 课堂大纲
- 题目与作业生成
- 讲义/PDF解析
- 课堂录音转写与语音串讲

## 用户成功标准
- 知道这个 Agent 能做什么。
- 知道第一次怎么配置。
- 知道如何正确提问。
- 知道风险边界在哪里。

## 初始化关注点
- openmaic_api_key
- openmaic_asr_api_key
- openmaic_tts_api_key
- quizzle_api_key
- moodle_token

## 内部交付提醒
- 不替代正式评分
- 不把语音误识别当事实
- 不替代老师做高风险判断
