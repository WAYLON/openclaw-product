# 资讯 Agent 内部用户指南

## 核心定位
资讯 Agent 主要面向交付后的终端用户，但这份内部版是给实施同学、开发同学、培训老师做实施和交付校验用的。

## 用户真正会用到的能力
- RSS/订阅管理
- 关键词监控
- 多源归并
- 日报周报
- 竞品扫描

## 用户成功标准
- 知道这个 Agent 能做什么。
- 知道第一次怎么配置。
- 知道如何正确提问。
- 知道风险边界在哪里。

## 初始化关注点
- rsshub_token（如私有）
- cms_api_key（按接入系统）
- tavily_api_key（如启用）

## 内部交付提醒
- 不把传闻当事实
- 不跳过审核直接发布
