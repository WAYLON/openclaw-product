# 资讯 Agent 内部快速启动

## 目标
这是给实施同学、开发同学、培训老师看的内部版本，用于确认这个 Agent 包已经被正确装配，并具备正式演示条件。

## 最短启动链路
1. 检查 `soul.yaml`、`AGENTS.md`、`IDENTITY.md`、`skills/` 是否齐全
2. 确认订阅源和CMS初始化状态
3. 绑定测试渠道实例
4. 运行首轮晨报/周报问题
5. 检查来源标记和待确认提示

## 首次测试建议
- 请整理今天 AI 行业晨报。
- 请监控三个关键词并合并重复内容。
- 请生成一份竞品周报框架。

## 重点检查项
- `soul.yaml` 是否已经切换成正式专业版。
- `AGENTS.md` 是否已写明资讯采集、归并、摘要原则。
- `IDENTITY.md` 是否明确资讯 Agent 的来源与风险边界。
- `skills/` 与 `docs/skills-matrix.md` 是否一致。
- 初始化缺失时是否先弹中文提示，而不是直接报错。
