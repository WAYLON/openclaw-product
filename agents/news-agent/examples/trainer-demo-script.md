# 资讯 Agent 讲师演示脚本

## 一、开场介绍
- 先说明 资讯 Agent 是平行独立 Agent，不依赖业务总控 Agent。
- 再说明它解决的核心问题：多源资讯归并、摘要、日报周报与发布准备。

## 二、初始化演示
- 先故意触发一次需要 Key 的能力。
- 让客户看到中文初始化提示。
- 再演示如何配置凭证并继续。

## 三、标准演示问题
- 请整理今天 AI 行业晨报。
- 请监控三个关键词并合并重复内容。
- 请生成一份竞品周报框架。

## 四、讲解顺序建议
1. 先讲资讯不是搜索而是归并
2. 再演示日报生成
3. 最后讲发布边界

## 五、风险边界提醒
- 不能把传闻当已验证事实
- 不能跳过审核直接发布高风险内容
