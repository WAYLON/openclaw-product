# 资讯 Agent 示例问法

## 标准问法
- 请先帮我说明你能处理什么，不能处理什么。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请帮我把今天的 AI 行业资讯整理成一份中文晨报，按政策、产品、资本、观点四栏输出。
- 请帮我监控“OpenClaw”“Agent 平台”“AI 助理”三个关键词，并合并重复资讯。
- 请把这周竞品公众号和行业媒体内容整理成一份对外可讲的周报框架。

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
