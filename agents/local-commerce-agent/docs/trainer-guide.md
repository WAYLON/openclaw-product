# 电商 / 本地生活 Agent 内部讲师指南

## 讲课目的
帮助实施老师快速理解这个 Agent 的讲法、演示顺序和风险边界，而不是临时拼凑话术。

## 建议授课流程
1. 先讲这个 Agent 解决什么问题。
2. 再讲它为什么必须独立安装、独立绑定渠道。
3. 再演示首次初始化。
4. 再跑一轮标准问题。
5. 最后讲风险边界和不适用场景。

## 建议演示问题
- 请写 3 版活动宣传文案。
- 请设计一个预约到店跟进流程。
- 请给商品图处理后说明文案。

## 讲师注意事项
- 不做虚假优惠承诺
- 不夸大库存权益
- 不跳过平台规则
