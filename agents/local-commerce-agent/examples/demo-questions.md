# 电商 / 本地生活 Agent 示例问法

## 标准问法
- 请先帮我说明你能处理什么，不能处理什么。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请帮我为这个门店活动写 3 版中文宣传文案，分别适合群发、私聊和海报。
- 请根据这批客户标签，设计一个预约到店跟进流程。
- 请把这组商品图先做背景去除，再给我一版电商详情页可用的说明文案。

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
