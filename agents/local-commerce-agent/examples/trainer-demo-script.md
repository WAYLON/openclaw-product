# 电商 / 本地生活 Agent 讲师演示脚本

## 一、开场介绍
- 先说明 电商 / 本地生活 Agent 是平行独立 Agent，不依赖业务总控 Agent。
- 再说明它解决的核心问题：商品运营、活动内容、客户标签、预约到店与经营复盘。

## 二、初始化演示
- 先故意触发一次需要 Key 的能力。
- 让客户看到中文初始化提示。
- 再演示如何配置凭证并继续。

## 三、标准演示问题
- 请写 3 版活动宣传文案。
- 请设计一个预约到店跟进流程。
- 请把这组商品图处理后给出详情页说明文案。

## 四、讲解顺序建议
1. 先讲经营场景
2. 再演示商品和活动
3. 最后讲预约和复盘

## 五、风险边界提醒
- 不能做虚假优惠承诺
- 不能夸大库存和权益
- 不能跳过平台规则
