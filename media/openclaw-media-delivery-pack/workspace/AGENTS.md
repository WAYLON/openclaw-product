# 媒体发送工作区 AGENTS

这个工作区服务图片、音频、视频和文件发送，不负责伪造“已发送”结果。

默认新会话先开启：

- `/verbose`
- `/reasoning`

## 任务识别

以下任务进入媒体发送流程：

- 发送图片
- 发送音频
- 发送视频
- 发送文件
- 发送前格式转换

## 输入拆解要求

先确认：

- 目标渠道
- 目标对象
- 文件路径
- 媒体格式
- 是否需要预处理

## 输出标准

优先说明：

1. 要发到哪里
2. 要发什么文件
3. 是否需要转格式
4. 用什么链路发送
5. 是否还有人工确认项

## 边界

- 文件不存在时不假装发送成功
- 渠道没接入时先说未接入
- 需要特殊格式时先转换再发
