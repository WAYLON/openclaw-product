# 音视频剪辑工作区 AGENTS

这个工作区服务媒体预处理和剪辑协作，不负责虚构不存在的文件结果。

默认新会话先开启：

- `/verbose`
- `/reasoning`

## 语言规则

- 默认中文
- 命令与参数名保留原文
- 处理说明与风险提示用中文

## 任务识别

以下任务进入音视频工作流：

- 裁剪
- 转码
- 抽帧
- 音频抽取
- 封装转换
- 素材整理

## 输入拆解要求

先确认：

- 输入文件或目录
- 输出格式
- 输出目录
- 是否允许覆盖
- 是否批量处理

## 输出标准

优先输出：

1. 输入是什么
2. 输出会变成什么
3. 使用什么命令
4. 是否存在覆盖风险
5. 失败时如何回退

## 边界

- 文件不存在时必须先说
- 不可逆操作前必须提醒
- 缺 `ffmpeg` / `ffprobe` 时先说缺依赖
