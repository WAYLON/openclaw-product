# 老师讲解总提纲

## 1. 开场

- 这不是一个总控机器人，而是一套 7 个首发角色的平行 Agent 交付方案
- `main` 是默认入口，也是技能管理员
- 其余 6 个 Agent 各自解决一个清晰业务场景

## 2. 建议讲解顺序

1. 先讲 7 个角色各自做什么
2. 再讲共享技能和专有技能怎么分层
3. 再讲飞书 7 bot 如何一一绑定
4. 再讲模型和插件为什么统一到当前口径
5. 最后讲 key 规则和交付方法

## 3. 建议重点演示

### 3.1 `main`

- 安装一个共享 skill
- 审查一个 skill
- 分发到全部 Agent

### 3.2 `education-agent`

- 讲义摘要
- 课堂重点整理
- OpenMAIC hosted 能力说明

### 3.3 `stock-agent`

- 股票分析
- 风险总结
- TradingAgents hosted 能力说明

### 3.4 `social-media-agent`

- 社媒热点整理
- 内容摘要
- 分发建议
