# 赠送技能清单

这个文档用于说明在不同产品版本中，默认赠送给客户的技能有哪些，以及这些技能分别起什么作用。

## 一、为什么要有“赠送技能清单”

很多客户会问：

- 我买这个包到底自带哪些能力？
- 哪些是基础功能？
- 哪些是增强能力？
- 哪些是后续加价项？

所以需要一份清晰的技能清单，而不是临时口头解释。

## 二、赠送技能分层

本项目建议把赠送技能分成三层：

1. 基础赠送技能
2. 行业增强技能
3. 客户定制技能

## 三、基础赠送技能

这些技能放在 `core/skills/`，属于所有产品共用的基础层，不再混同于股票行业技能。

当前默认基础层采用固定 10 技能预设：

1. `tavily-search`
2. `find-skills`
3. `proactive-agent-1-2-4`
4. `agent-browser`
5. `remotion-best-practices`
6. `cron`
7. `planning-with-files`
8. `baoyu-skills`
9. `anthropics/skills`
10. `aliyun-tongyi-wanxiang`

对应文件：

- `core/skills/basic-10.lock.json`
- `core/scripts/install-core-skills.sh`
- `core/scripts/check-core-skills.sh`

这 10 个技能的定位是：

- 联网搜索
- 技能发现
- 主动巡检
- 浏览器自动化
- 视频生成
- 定时任务
- 长任务规划
- 内容创作
- 官方技能扩展
- 国产图像生成

## 四、行业增强技能

这些技能是叠加在基础层之上的行业附加技能。

### 1. `stock-research-core`

作用：

- 股票研究总控技能
- 把模糊问题拆成结构化任务
- 统一股票研究输出格式

### 2. `qveris-official`

作用：

- 发现外部 API
- 调用结构化外部能力
- 作为行业包中的能力补充

### 3. `jqdata-playbook`

作用：

- 聚宽 / JQData 研究工作流层
- 约束聚宽方向的研究结构
- 提供聚宽文档和使用规则

适合：

- A 股历史研究
- 分钟级研究
- 回测规划
- 聚宽文档型支持

### 4. `jqdata-research`

作用：

- 真正执行聚宽 / A 股历史研究
- 支持脚本化研究、筛选与回测

适合：

- 需要真实历史数据执行的客户

### 5. `ths-ifind-official`（建议后续补充）

作用：

- 同花顺 / iFinD 盘中选股
- 字段映射
- 盘中筛选与行情支撑

适合：

- 盘中选股
- 实时市场观察
- 同花顺终端用户

## 五、客户定制技能

这些技能不是通用版默认赠送，而是跟客户项目强绑定。

例如：

- 客户专属工作流 skill
- 客户专属数据接口 skill
- 客户专属 SaaS 接入 skill
- 客户专属自动化脚本 skill

这类能力通常只会出现在：

- 客户专版
- 团队版

## 六、当前项目中的推荐赠送方案

### 通用股票基础版

对应：

- `core/skills/basic-10.lock.json`
- `stocks/openclaw-stock-pack`

建议默认赠送：

- 基础 10 技能预设
- `stock-research-core`
- `qveris-official`

### LangLang 股票专用版

对应：

- `core/skills/basic-10.lock.json`
- `stocks/openclaw-stock-pack-langlang`

建议默认赠送：

- 基础 10 技能预设
- `stock-research-core`
- `qveris-official`
- `jqdata-playbook`

建议视目标机器环境追加：

- `jqdata-research`

建议后续补齐：

- `ths-ifind-official`

## 七、不建议默认赠送的技能

这些能力不建议一上来默认送给所有客户：

- `find-skills`
- `clawhub`
- 自动安装外部技能的能力
- 来源不明的第三方技能
- 高危执行类技能

原因：

- 增加 Token 消耗
- 增加安全风险
- 增加跑偏概率

## 八、怎么对客户介绍

建议不要说：

- “我们送你一堆技能”

建议说：

- “我们会赠送一组基础必备技能”
- “增强能力按行业和客户场景分层配置”
- “客户专属能力按定制版交付”

## 九、一句话总结

赠送技能的目的，不是让技能看起来很多，而是让客户一上来就拥有最必要、最稳定、最能体现价值的能力。
