# 企业级优化说明

> 本说明用于解释当前项目相对最初骨架版，已经补上的企业级工程能力。

## 1. 当前新增的企业级能力

### 1.1 工程元数据

- 增加了 [pyproject.toml](/Users/waylon/Desktop/openclaw-product/pyproject.toml)
- 明确了：
  - Python 版本要求
  - 运行依赖

### 1.2 依赖与忽略规则

- 增加了 [requirements.txt](/Users/waylon/Desktop/openclaw-product/requirements.txt)
- 增加了 [.gitignore](/Users/waylon/Desktop/openclaw-product/.gitignore)
- 增加了标准 `.venv` 运行方式，通过 [Makefile](/Users/waylon/Desktop/openclaw-product/Makefile) 统一管理本地依赖

### 1.3 统一工程入口

- 增加了 [Makefile](/Users/waylon/Desktop/openclaw-product/Makefile)
- 支持：
  - `make bootstrap`
  - `make doctor`
  - `make quality`
  - `make install-all`

### 1.4 质量检查

- 增加了 [quality_check.py](/Users/waylon/Desktop/openclaw-product/scripts/quality_check.py)
- 当前会检查：
  - 关键工程文件是否存在
  - Python 语法是否通过
  - Agent 模板数量
  - 交付包 trainer-guide 覆盖率

### 1.5 配置结构校验

- 增加了 [schema_validator.py](/Users/waylon/Desktop/openclaw-product/core/config/schema_validator.py)
- 当前已能校验：
  - `install-manifest.yaml`

### 1.6 安装前校验

- 安装器现在不再是“目录存在就安装”
- [agent_installer.py](/Users/waylon/Desktop/openclaw-product/installer/agent_installer.py) 会先调用严格校验
- 包结构或 Agent 结构不完整时，会在安装前拦截
