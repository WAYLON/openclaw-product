# 版本发布流程

## 1. 发布前检查

执行：

```bash
make quality
python3 scripts/check_all_packages.py
python3 installer/bootstrap_project.py check
./agent-platform doctor
```

## 2. 文档检查

- 检查 8 个 Agent 的交付文档是否完整
- 检查安装文档是否与当前命令一致
- 检查渠道绑定总表是否与当前策略一致
- 检查模型分配总表是否与 `agent.yaml` 一致

## 3. 交付包检查

- 每个 `packages/<agent>/config/install-manifest.yaml` 必须存在
- 每个包必须包含：
  - `quickstart.md`
  - `user-guide.md`
  - `trainer-guide.md`
  - `faq.md`
  - `examples.md`

## 4. 本机 OpenClaw 联调检查

- 至少导出 2 个 Agent 到本机 OpenClaw
- 至少验证 1 个飞书实例绑定
- 至少跑通 1 次最小请求：
  - `openclaw agent --agent <agent> --message "回复 OK" --json`

## 5. 发布建议

- 小版本：文档、模板、校验增强
- 中版本：安装流程、渠道绑定、模型策略调整
- 大版本：目录结构、Agent 结构、权限模型变化
