# 专有技能安装示例

这份文档保留两个已经真实接通过的专有技能示例：

- `education-agent -> openmaic`
- `stock-agent -> tradingagents-analysis`

## 一、education-agent -> openmaic

### 1. 注册 Agent

```bash
openclaw agents add education-agent
```

### 2. 安装项目技能包并同步

```bash
cd ~/Desktop/openclaw-product
./agent-platform agent install education-agent
./agent-platform openclaw export education-agent
```

### 3. 绑定飞书

```bash
openclaw agents bind --agent education-agent --channel feishu --account educationbot
```

### 4. 配置 OpenMAIC hosted access code

写入：

- `~/.openclaw/openclaw.json`

配置路径：

- `skills.entries.openmaic.config.accessCode`

### 5. 可见性要求

如果现场是把 `openmaic` 按教育 Agent 私有 skill 方式接入，需要额外确认：

1. skill 放在：
   - `~/.openclaw/workspaces/education-agent/skills/openmaic/`
2. `SKILL.md` 带合法 frontmatter
3. 拷贝后已重启 gateway
4. 用新会话验证，不用既有线程结果代替
5. 如果仍沿用原有技能清单，直接重建 `education-agent` 的 session snapshot

### 6. 已跑通的现场事实

- 私有 skill 路径按下面这条实际运行：
  - `~/.openclaw/workspaces/education-agent/skills/openmaic/SKILL.md`
- `references/hosted-mode.md` 已补齐
- `references/generate-flow.md` 已补齐
- `accessCode` 已通过：
  - `https://open.maic.chat/api/health`
- `education-agent` 已成功提交真实课堂生成任务，不是只回大纲

现场实测任务示例：

- `Job ID: 4iOKux0RvQ`
- 运行状态：`running / generating_scenes`
- 说明：已经进入真实生成链路

### 7. 最小验收

```bash
openclaw agent --agent education-agent --message "请只回复ok"
```

如果刚接入或刚修过 `openmaic`，建议直接开新会话再测：

```bash
openclaw agent --agent education-agent --message "帮我把这份讲义整理成课堂重点"
```

标准演示语句：

```text
用 openmaic 生成一个互动课堂：零基础文科生，30 分钟学会 Python。生成完成后直接把 classroom URL 发我。
```

## 二、stock-agent -> tradingagents-analysis

### 1. 前置条件

先按官方方式安装最新稳定版 OpenClaw：

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
openclaw onboard --install-daemon
openclaw --version
openclaw status
```

### 2. 注册 stock-agent

```bash
openclaw agents add stock-agent
openclaw agents list
```

### 3. 初始化项目状态并同步

```bash
cd ~/Desktop/openclaw-product
./agent-platform init
./agent-platform start
./agent-platform doctor
./agent-platform agent install stock-agent
./agent-platform openclaw export stock-agent
./agent-platform agent list
```

### 4. 确认官方工作区已生成

这一步由官方 `openclaw agents add stock-agent` 生成：

- `~/.openclaw/workspaces/stock-agent/soul.yaml`
- `~/.openclaw/workspaces/stock-agent/AGENTS.md`
- `~/.openclaw/workspaces/stock-agent/IDENTITY.md`

### 5. 绑定渠道

飞书示例：

```bash
openclaw agents bind --agent stock-agent --channel feishu --account stockbot
```

### 6. 配置股票相关 key

当前关键项：

- `TRADINGAGENTS_TOKEN`
- `QVERIS_API_KEY`

后台运行时 key 统一写：

- `~/Library/LaunchAgents/ai.openclaw.gateway.plist`

### 7. 检查技能落点

```bash
find agents/stock-agent/skills -maxdepth 2 -type f | sort
find .platform/workspaces/stock-agent/skills -maxdepth 2 -type f | sort
find ~/.openclaw/workspaces/stock-agent/skills -maxdepth 2 -type f | sort
```

### 8. 首次联调建议

建议先测试：

- 帮我解析一下宁德时代的股票代码和市场
- 帮我做一份最近 30 天的技术面摘要
- 帮我生成一份中文研究简报

最小验收：

```bash
openclaw agent --agent stock-agent --message "请用一句话介绍你自己"
```

### 9. 风险边界

股票 Agent 当前定位是：

- 研究辅助
- 风险提示
- 信息归纳

当前不建议直接作为默认首发能力上线：

- 自动交易
- 直接荐股
- 自动执行买卖

### 10. 失败排查

如果渠道绑定后无响应，先确认：

```bash
openclaw agents bindings --json
```

看是否存在：

```text
stock-agent <- feishu:stockbot
```
