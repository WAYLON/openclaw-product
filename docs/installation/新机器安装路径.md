# 新机器安装路径

> 适用于一台还没有安装过这套环境的新机器。

## 原则

- OpenClaw 本体安装优先走官方文档
- 飞书插件安装优先走官方文档
- 本项目只补 7 个 Agent、插件、技能和交付结构
- 如果机器上曾装过旧环境，先改走：
  - [重装与清理路径.md](/Users/waylon/Desktop/openclaw-product/docs/installation/重装与清理路径.md)

## 推荐顺序

1. 按官方文档安装最新稳定版 OpenClaw
2. 按官方文档安装飞书官方插件
3. 初始化本机 OpenClaw
4. 配置 `local-proxy/gpt-5.4`
5. 启用：
   - `memory-lancedb-pro`
   - `lossless-claw`
6. 安装 6 个业务 Agent 模板
7. 同步到本机 OpenClaw
8. 绑定 7 个 bot
9. 写 key
10. 执行：
   - `make delivery-check`
   - `./agent-platform acceptance --full --report ./reports/new-machine-acceptance.md`

## 项目命令

```bash
make venv
./agent-platform init
./agent-platform doctor
./agent-platform agent install education-agent
./agent-platform agent install stock-agent
./agent-platform agent install loan-agent
./agent-platform agent install social-media-agent
./agent-platform agent install news-agent
./agent-platform agent install sales-agent
./agent-platform openclaw export-all --openclaw-home ~/.openclaw
make delivery-check
make acceptance
```
