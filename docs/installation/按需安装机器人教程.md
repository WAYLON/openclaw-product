# 按需安装机器人教程

> 适用场景：客户不需要一次安装 8 个机器人，只想先安装其中 1 到 3 个做试点。  
> 推荐原则：**先小范围安装、先小范围验证、再逐步扩容。**

## 1. 先决定本次要装哪几个

常见的首装组合：

- 教育场景：`education-agent`
- 股票研究场景：`stock-agent`
- 修图服务场景：`photo-polisher`
- 电商 / 本地生活场景：`local-commerce-agent`
- 客服销售场景：`sales-agent`
- 法务合同场景：`legal-agent`

如果客户还没想清楚，建议首期只装：

- `education-agent`
- `stock-agent`
- `photo-polisher`

这 3 个最容易演示，业务边界也清楚。

## 2. 安装前提

先按官方方式安装 OpenClaw：

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
openclaw onboard --install-daemon
openclaw status
```

预期：

- `Gateway reachable`
- `LaunchAgent ... running`

如果这里不过，先不要继续装 Agent。

## 3. 只安装指定的几个 Agent

下面以安装 3 个机器人为例：

- `education-agent`
- `stock-agent`
- `photo-polisher`

### 3.1 官方注册 Agent

```bash
openclaw agents add education-agent
openclaw agents add stock-agent
openclaw agents add photo-polisher
```

### 3.2 进入项目目录，安装模板

```bash
cd ~/Desktop/openclaw-product
./agent-platform init
./agent-platform agent install education-agent
./agent-platform agent install stock-agent
./agent-platform agent install photo-polisher
```

### 3.3 同步到本机 OpenClaw

```bash
./agent-platform openclaw export education-agent
./agent-platform openclaw export stock-agent
./agent-platform openclaw export photo-polisher
```

### 3.4 设置 identity

```bash
openclaw agents set-identity \
  --agent education-agent \
  --identity-file /Users/waylon/.openclaw/parallel-agents/workspaces/education-agent/IDENTITY.md

openclaw agents set-identity \
  --agent stock-agent \
  --identity-file /Users/waylon/.openclaw/parallel-agents/workspaces/stock-agent/IDENTITY.md

openclaw agents set-identity \
  --agent photo-polisher \
  --identity-file /Users/waylon/.openclaw/parallel-agents/workspaces/photo-polisher/IDENTITY.md
```

## 4. 验证是否真的只装了这几个

```bash
openclaw agents list
```

如果你本机之前很干净，预期至少会看到：

- `main`
- `education-agent`
- `stock-agent`
- `photo-polisher`

再做最小调用测试：

```bash
openclaw agent --agent education-agent --message "请用一句话介绍你自己" --json
openclaw agent --agent stock-agent --message "请用一句话介绍你自己" --json
openclaw agent --agent photo-polisher --message "请用一句话介绍你自己" --json
```

## 5. 后续想加装其他机器人怎么办

按同样方式继续装，不需要重装整套。

例如后面再补装法务机器人：

```bash
openclaw agents add legal-agent
cd ~/Desktop/openclaw-product
./agent-platform agent install legal-agent
./agent-platform openclaw export legal-agent
openclaw agents set-identity \
  --agent legal-agent \
  --identity-file /Users/waylon/.openclaw/parallel-agents/workspaces/legal-agent/IDENTITY.md
```

这就是“按需安装”的核心原则：

- 先装需要的
- 不需要的先不装
- 后续再追加

## 6. 如果客户只要 1 个机器人

例如客户只要教育机器人，最短命令就是：

## 8. 常见误区

### 误区 1：一定要一次装 8 个

不需要。  
这个项目支持按需安装，装几个都可以。

### 误区 2：后面加机器人要重装全部

不需要。  
后续只要对新增机器人重复：

- `openclaw agents add`
- `./agent-platform agent install`
- `./agent-platform openclaw export`
- `openclaw agents set-identity`

### 误区 3：项目里安装了就等于 OpenClaw 里装好了

不等于。  
项目里安装的是模板，真正给本机 OpenClaw 用，还要执行：

```bash
./agent-platform openclaw export <agent>
```

## 9. 推荐给客户的实施策略

建议按下面顺序推进：

1. 先装 1 到 2 个最关键机器人
2. 先接一个最常用渠道
3. 先做 1 天到 3 天小范围试运行
4. 试运行稳定后，再补装剩余机器人

如果客户业务还没定型，不建议第一次就把 8 个全部上齐。
