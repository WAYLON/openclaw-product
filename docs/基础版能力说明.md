# 基础版能力说明

这个文档用于说明当前基础版产品到底由哪几层组成，以及这些技能分别怎么使用。

## 一、什么是基础版

在当前仓库中，基础版不再等于某一个股票目录，而是：

1. `core/`
2. `stocks/openclaw-stock-pack`

它的定位是：

- 先安装 OpenClaw 本体
- 先安装一套通用基础技能
- 通用股票研究
- 基础选股支持
- 新闻判断
- 结构化研究路由
- 外部能力补充

它不包含聚宽 / JQData 专用能力。

## 二、基础版包含哪些技能层

当前基础版包含两层技能：

### 第一层：core 基础技能预设

对应：

- `core/skills/basic-10.lock.json`

当前固定为 10 个默认技能：

- `tavily-search`
- `find-skills`
- `proactive-agent-1-2-4`
- `agent-browser`
- `remotion-best-practices`
- `cron`
- `planning-with-files`
- `baoyu-skills`
- `anthropics/skills`
- `aliyun-tongyi-wanxiang`

它们负责：

- 联网搜索
- 技能发现
- 主动任务
- 浏览器自动化
- 定时任务
- 长任务规划
- 内容创作
- 图像与视频能力

补充说明：

- 这 10 个是内部技术安装预设
- 不代表正式交付给客户后要把所有发现类、自动安装类能力长期放开
- `find-skills` 一类能力默认只建议安装人员调试时使用

安装入口：

- `core/scripts/install-core-skills.sh`
- `core/scripts/check-core-skills.sh`

### 第二层：股票行业附加技能

#### 1. `stock-research-core`

作用：

- 识别当前是不是股票研究任务
- 把模糊问题拆成结构化研究任务
- 路由到合适的能力
- 统一输出格式

路径：

- `stocks/openclaw-stock-pack/skills/stock-research-core/`

#### 2. `qveris-official`

作用：

- 发现外部 API
- 调用结构化外部能力
- 作为本地技能不足时的能力补充

路径：

- `stocks/openclaw-stock-pack/skills/qveris-official/`

## 三、基础版不包含哪些能力

基础版当前不包含：

- `jqdata-playbook`
- 聚宽专版文档层
- `jqdata-research`
- 聚宽最小验证脚本

如果客户需要这些能力，应使用：

- `stocks/openclaw-stock-pack-langlang`

或未来其他聚宽增强版。

## 四、基础版怎么用

### 用法 1：先使用 core 基础层能力

例如：

- “帮我联网查一下最新政策”
- “帮我用浏览器打开这个页面并提取要点”
- “帮我设置一个定时提醒”

这时主要会用到 core 基础技能预设。

### 用法 2：作为通用股票研究助手

直接在会话里提问：

- “帮我从股票研究角度梳理一下这个问题”
- “帮我设计一个盘中突破观察框架”
- “帮我判断这条消息对哪个板块影响更大”

这时主要会用到股票行业附加技能中的：

- `stock-research-core`

### 用法 3：作为外部能力发现入口

如果本地没有相关能力，可以提问：

- “帮我发现一个适合查结构化数据的外部 API”
- “如果本地做不到，优先用 QVeris 找能力”

这时主要会用到：

- `qveris-official`

### 用法 4：作为轻量版行业工作区

如果采用 `full` 安装模式，基础版还会带来：

- 股票研究模式的 `SOUL`
- 股票任务的 `AGENTS`
- 基础工作区模板

适合：

- 客户想快速上手
- 客户暂时不需要聚宽这类重型能力

## 五、建议的安装方式

### 先装基础技能

```bash
./core/scripts/install-core-skills.sh
```

### 再安装股票行业包

适合已有自己工作区规则的客户：

```bash
./scripts/install-pack.sh stocks/openclaw-stock-pack skills-only
```

### 安装完整工作区

适合愿意采用这套股票研究模式的客户：

```bash
./scripts/install-pack.sh stocks/openclaw-stock-pack full
```

## 六、基础版适合什么客户

适合：

- 第一次接触 OpenClaw 的客户
- 想先体验股票方向能力的客户
- 不想一上来接入聚宽的客户
- 想先从轻量版开始，再升级的客户

## 七、基础版的价值

基础版的价值不在“功能很多”，而在于：

- 基础层与行业层分离
- 够轻
- 够清楚
- 容易交付
- 容易理解
- 容易作为后续升级入口

## 八、一句话总结

基础版不是“缩水版”，而是：

**最适合作为引流、试用、低门槛交付的通用行业包。**
