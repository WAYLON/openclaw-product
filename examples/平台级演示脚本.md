# 平台级演示脚本

## 演示目标
向客户说明：这不是一个总控机器人，而是一套可安装、可配置、可讲解、可交付的平行 Agent 平台。

## 建议演示顺序
1. 先展示平台目录结构和 8 个专业 Agent 的平行关系。
2. 再展示平台初始化、Agent 安装和渠道绑定命令。
3. 再进入任意一个专业包，演示 quickstart、user-guide 和 trainer-guide 的完整交付。
4. 最后演示首次命中 key 时的中文初始化提示。
