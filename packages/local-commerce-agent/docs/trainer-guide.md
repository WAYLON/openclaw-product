# 电商 / 本地生活 Agent 讲师讲解手册

> 这是正式交付物，面向老师、实施顾问、售前顾问。

## 讲解目标
- 让客户理解这个 Agent 解决什么问题。
- 让客户知道首次怎么装、怎么配、怎么测。
- 让客户知道风险边界与最佳实践。

## 推荐讲解顺序
1. 先讲经营场景
2. 再演示商品和活动
3. 最后讲预约和复盘

## 讲师话术模板
- 这不是一个总控机器人，而是一个独立专业 Agent。
- 它有自己的灵魂、提示词、技能绑定、记忆与文档。
- 它主要解决的是：商品运营、活动内容、客户标签、预约到店与经营复盘。
- 如果第一次调用外部能力缺少 Key，它不会直接报错，而会先提示你如何初始化。

## 标准演示脚本
- 请写 3 版活动宣传文案。
- 请设计一个预约到店跟进流程。
- 请把这组商品图处理后给出详情页说明文案。

## 重点风险边界
- 不能做虚假优惠承诺
- 不能夸大库存和权益
- 不能跳过平台规则
