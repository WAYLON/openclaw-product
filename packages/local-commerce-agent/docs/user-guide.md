# 电商 / 本地生活 Agent 用户指南

## 1. 用户能获得什么
- 商品目录查询
- 活动文案生成
- FAQ 回答
- 预约到店流程
- 商品图清理与复盘简报

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认门店或商品场景
2. 补齐内容/CRM/预约相关 Key
3. 演示文案与预约
4. 再讲复盘指标

## 4. 什么时候需要先初始化 Key
- cms_api_key
- espocrm_api_key
- chatwoot_api_key
- calcom_api_key
- n8n_api_key

## 5. 风险与边界
- 不能做虚假优惠承诺
- 不能夸大库存和权益
- 不能跳过平台规则
