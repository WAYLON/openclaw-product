# 电商 / 本地生活 Agent 示例问法

## 标准问法
- 请先说明当前哪些能力可直接使用，哪些需要额外接入。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请写 3 版活动宣传文案。
- 请设计一个预约到店跟进流程。
- 请把这组商品图处理后给出详情页说明文案。

## 结构化示例
```text
结论：先约到店
转化优先级：高
判断依据：
- 用户意图：对方多次询问使用流程和到店时间
- 场景适配：当前更适合线下体验后成交
- 资源条件：门店有可预约时段，接待能力充足
说明：当前直接推活动不如先把到店动作定下来，更容易转化。
下一步建议：先发送预约话术和到店说明，再视反馈补充活动信息。
```

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
- 让你越过专业边界直接替别人做决定。
