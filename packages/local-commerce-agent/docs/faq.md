# 电商 / 本地生活 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，或当前请求依赖商品目录、CRM、预约或图片处理外部能力尚未接入。

## Q2：为什么文档里提到的经营系统能力没直接可用？
A：这些能力依赖第三方平台接入；当前未接通时，只保留文案、问答和流程说明能力。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：为什么不能直接替我定价？
A：因为这个 Agent 是经营助手，不替商家做最终商业决策。

## Q6：为什么会强调到店流程？
A：因为本地生活场景里，预约和到店体验本身就是转化关键环节。
