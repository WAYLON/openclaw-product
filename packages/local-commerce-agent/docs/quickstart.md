# 电商 / 本地生活 Agent Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
电商 / 本地生活 Agent 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：商品运营、活动内容、客户标签、预约到店与经营复盘。

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 电商 / 本地生活 Agent 包。
3. 绑定一个渠道实例到当前 Agent。
4. 检查并初始化必要凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请写 3 版活动宣传文案。
- 请设计一个预约到店跟进流程。
- 请把这组商品图处理后给出详情页说明文案。

## 4. 首次讲解重点
1. 先讲经营场景
2. 再演示商品和活动
3. 最后讲预约和复盘
