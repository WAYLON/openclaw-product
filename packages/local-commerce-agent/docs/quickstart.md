# 电商 / 本地生活 Agent Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
电商 / 本地生活 Agent 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：商品运营、活动内容、客户标签、预约到店与经营复盘。

## 1.1 当前交付边界
- 当前可直接验证的是：中文活动文案、FAQ 与预约流程说明。
- 商品目录、CRM、客服台、预约系统、经营复盘等能力，需要额外配置或第三方平台接入。
- 文档里的外部经营系统名称，仅表示接入方向，不默认视为当前已打通。

## 1.2 判断结果结构
- 结论
- 转化优先级
- 判断依据
  - 用户意图
  - 场景适配
  - 资源条件
- 说明
- 下一步建议

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 电商 / 本地生活 Agent 包。
3. 绑定一个渠道实例到当前 Agent。
4. 如需启用经营系统或图片处理等外部能力，再检查并初始化相关凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请写 3 版活动宣传文案。
- 请设计一个预约到店跟进流程。
- 请把这组商品图处理后给出详情页说明文案。

## 4. 首次讲解重点
1. 先讲经营场景
2. 再演示商品和活动
3. 最后讲预约和复盘
