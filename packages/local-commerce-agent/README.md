# 电商本地生活专业包

这是 `电商 / 本地生活 Agent` 的正式交付包。

## 包定位
- 主要解决：商品运营、活动内容、客户标签、预约到店与经营复盘
- 交付对象：管理员、实施顾问、培训讲师、终端用户
- 交付目标：可安装、可演示、可教学、可验证

## 当前交付边界
- 当前可直接交付的是：中文活动文案、经营问答、流程说明、基础商品与门店内容输出。
- 商品目录、CRM、客服台、预约系统、经营复盘等能力需要额外完成外部配置或第三方平台接入。
- 文档提到的 Medusa、Saleor、EspoCRM、Chatwoot、Cal.com、n8n 等来源，不代表当前环境已默认打通。

## 判断型输出结构
- 结论
- 转化优先级
- 判断依据
  - 用户意图
  - 场景适配
  - 资源条件
- 说明
- 下一步建议

## 包内包含
- quickstart
- user-guide
- trainer-guide
- faq
- examples
- install-manifest
- image-plan

## 交付原则
- 中文优先
- 讲师可直接讲
- 管理员可直接安装
- 用户可直接上手
- 风险边界必须可见
