# 资讯 Agent 讲师讲解手册

> 这是正式交付物，面向老师、实施顾问、售前顾问。

## 讲解目标
- 让客户理解这个 Agent 解决什么问题。
- 让客户知道首次怎么装、怎么配、怎么测。
- 让客户知道风险边界与最佳实践。

## 推荐讲解顺序
1. 先讲资讯不是搜索而是归并
2. 再演示日报生成
3. 最后讲发布边界

## 讲师话术模板
- 这不是一个总控机器人，而是一个独立专业 Agent。
- 它有自己的灵魂、提示词、技能绑定、记忆与文档。
- 它主要解决的是：多源资讯归并、摘要、日报周报与发布准备。
- 如果第一次调用外部能力缺少 Key，它不会直接报错，而会先提示你如何初始化。

## 标准演示脚本
- 请整理今天 AI 行业晨报。
- 请监控三个关键词并合并重复内容。
- 请生成一份竞品周报框架。

## 重点风险边界
- 不能把传闻当已验证事实
- 不能跳过审核直接发布高风险内容
