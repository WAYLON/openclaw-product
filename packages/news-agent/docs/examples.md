# 资讯 Agent 示例问法

## 标准问法
- 请先说明当前哪些能力可直接使用，哪些需要额外接入。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请整理今天 AI 行业晨报。
- 请监控三个关键词并合并重复内容。
- 请生成一份竞品周报框架。

## 结构化示例
```text
结论：值得保留并纳入简报
信息可信度：中
判断依据：
- 来源可靠性：来自主流媒体和企业公告，能相互印证
- 信息时效性：发布时间在 24 小时内
- 重复与噪声：多条内容在讲同一件事，已去重整理
说明：这条信息有传播价值，但仍建议标注“待持续跟踪”。
下一步建议：纳入晨报正文，并继续观察是否有后续官方补充。
```

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
- 让你越过专业边界直接替别人做决定。
