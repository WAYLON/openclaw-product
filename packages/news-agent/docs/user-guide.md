# 资讯 Agent 用户指南

## 1. 用户能获得什么
- 当前可直接使用：中文资讯整理、主题归并、日报周报框架
- 按接入情况可启用：订阅源管理、关键词监控、竞品扫描、发布准备

## 1.1 当前交付状态说明
- 已内置：摘要、归并、中文简报输出
- 需外部配置：联网增强、私有订阅源接入
- 需第三方平台：CMS 发布、内容后台同步
- 仅规划：未单独交付的文档导出与发布链路，不作为默认开箱承诺

## 1.2 判断结果输出格式
- 结论：值得保留 / 继续关注 / 暂缓引用
- 信息可信度：高 / 中 / 低
- 判断依据：
  - 来源可靠性：是否来自官方或稳定来源、能否交叉验证、发布者是否长期可信
  - 信息时效性：是不是新消息、是否仍有传播价值、是否已有更新版本
  - 重复与噪声：是否重复转载、是否标题夸张、是否缺少新增事实
- 说明：用简短中文说明取舍理由
- 下一步建议：继续跟踪、补证据、纳入简报，或暂不引用

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认主题与来源范围
2. 补齐订阅或 CMS 凭证
3. 生成摘要与聚类结果
4. 决定是否进入发布流程

## 4. 什么时候需要先初始化 Key
- 只有在你要启用私有订阅源、联网增强或 CMS 发布时，才需要初始化对应凭证。
- 当前交付若仅做中文资讯整理与简报输出，可先不配置这些外部 Key。

## 5. 风险与边界
- 不能把传闻当已验证事实
- 不能跳过审核直接发布高风险内容
