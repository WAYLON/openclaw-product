# 资讯 Agent 用户指南

## 1. 用户能获得什么
- RSS 与订阅管理
- 关键词监控
- 多源归并
- 日报/周报生成
- 竞品内容扫描

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认主题与来源范围
2. 补齐订阅或 CMS 凭证
3. 生成摘要与聚类结果
4. 决定是否进入发布流程

## 4. 什么时候需要先初始化 Key
- rsshub_token（如私有）
- cms_api_key（Strapi/Directus/Payload 按需）
- tavily_api_key（如使用联网增强）

## 5. 风险与边界
- 不能把传闻当已验证事实
- 不能跳过审核直接发布高风险内容
