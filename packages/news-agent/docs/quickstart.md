# 资讯 Agent Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
资讯 Agent 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：多源资讯归并、摘要、日报周报与发布准备。

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 资讯 Agent 包。
3. 绑定一个渠道实例到当前 Agent。
4. 检查并初始化必要凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请整理今天 AI 行业晨报。
- 请监控三个关键词并合并重复内容。
- 请生成一份竞品周报框架。

## 4. 首次讲解重点
1. 先讲资讯不是搜索而是归并
2. 再演示日报生成
3. 最后讲发布边界
