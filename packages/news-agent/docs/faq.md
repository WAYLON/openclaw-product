# 资讯 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，系统会先提示你配置缺失凭证。

## Q2：为什么不让它顺手做别的业务？
A：因为每个 Agent 都是平行独立关系，不设计业务总控 Agent。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：为什么会标注“待确认”？
A：因为资讯 Agent 必须区分已验证信息和待核实内容。

## Q6：为什么同一条新闻被合并了？
A：因为系统优先做去重和聚类，避免把重复内容当多条独立发现。
