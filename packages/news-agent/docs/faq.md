# 资讯 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，或当前请求依赖外部订阅源、联网增强或 CMS 平台尚未接入。

## Q2：为什么文档里提到的发布链路没直接可用？
A：发布链路依赖 CMS 或自动化平台接入，当前未接通时不应被理解为默认可用。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：为什么会标注“待确认”？
A：因为资讯 Agent 必须区分已验证信息和待核实内容。

## Q6：为什么同一条新闻被合并了？
A：因为系统优先做去重和聚类，避免把重复内容当多条独立发现。
