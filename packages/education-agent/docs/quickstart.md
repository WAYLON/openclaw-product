# 教育 Agent Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
教育 Agent 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：备课、题目、讲义解析、语音课堂接入与老师讲解交付。

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 教育 Agent 包。
3. 绑定一个渠道实例到当前 Agent。
4. 检查并初始化必要凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请设计一节函数图像课的大纲。
- 请把这段课堂录音整理成中文纪要。
- 请把这份讲义改成老师可播报的语音串讲稿。

## 4. 首次讲解重点
1. 先讲教学设计能力
2. 再演示课堂资料解析
3. 最后演示语音转写或播报
