# 教育 Agent 示例问法

## 标准问法
- 请先说明当前哪些能力可直接使用，哪些需要额外接入。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请设计一节函数图像课的大纲。
- 请把这段课堂录音整理成中文纪要。
- 请把这份讲义改成老师可播报的语音串讲稿。

## 结构化示例
```text
结论：更适合讲授后再安排练习
优先级：高
判断依据：
- 学习目标：本节课重点是先让学生理解函数图像概念
- 学生阶段：当前学段适合先建立图像直观认识
- 课堂资源：有完整讲义，但练习时间有限
说明：先讲授再安排少量练习，更容易保证学生先听懂核心概念。
下一步建议：先输出讲课提纲，再补 3 到 5 道基础练习题。
```

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
- 让你越过专业边界直接替别人做决定。
