# 教育 Agent 示例问法

## 标准问法
- 请先帮我说明你能处理什么，不能处理什么。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请设计一节函数图像课的大纲。
- 请把这段课堂录音整理成中文纪要。
- 请把这份讲义改成老师可播报的语音串讲稿。

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
- 让你越过专业边界直接替别人做决定。
