# 教育 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，或当前请求依赖语音/课堂平台能力尚未接入。系统会先提示缺少什么。

## Q2：为什么文档里提到的语音或课堂能力没直接跑？
A：这些能力通常需要额外的 ASR/TTS 配置或第三方教学平台，不作为默认开箱承诺。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：教育 Agent 支持语音吗？
A：支持，当前已覆盖课堂录音转写、语音播报和口语练习点评。

## Q6：为什么上传语音后没继续？
A：通常是缺少 ASR/TTS Key，或当前渠道没有透传语音文件。
