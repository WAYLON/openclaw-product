# 教育 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，系统会先提示你配置缺失凭证。

## Q2：为什么不让它顺手做别的业务？
A：因为每个 Agent 都是平行独立关系，不设计业务总控 Agent。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：教育 Agent 支持语音吗？
A：支持，当前已覆盖课堂录音转写、语音播报和口语练习点评。

## Q6：为什么上传语音后没继续？
A：通常是缺少 ASR/TTS Key，或当前渠道没有透传语音文件。
