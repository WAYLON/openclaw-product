# 教育 Agent 用户指南

## 1. 用户能获得什么
- 课程主题规划
- 课堂大纲生成
- 题目与作业生成
- 课件/PDF 解析
- 课堂录音转写与语音串讲

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认学科、学段、课时长度
2. 补齐教学与语音相关 Key
3. 演示课堂设计
4. 再演示语音能力

## 4. 什么时候需要先初始化 Key
- openmaic_api_key
- openmaic_asr_api_key
- openmaic_tts_api_key
- quizzle_api_key
- moodle_token

## 5. 风险与边界
- 不替代正式评分
- 不把语音误识别内容当事实
- 不替代老师做高风险学生判断
