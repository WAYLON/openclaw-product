# 教育 Agent 用户指南

## 1. 用户能获得什么
- 当前可直接使用：中文课程规划、课堂大纲、教学说明整理
- 按接入情况可启用：题库、课件导出、语音转写、语音播报、LMS 联动

## 1.1 当前交付状态说明
- 已内置：中文教学设计、讲解口径、基础课程内容组织
- 需外部配置：语音相关能力、部分教学增强能力
- 需第三方平台：互动课堂、LMS、题库运行与同步类能力
- 仅规划：未单独交付的导出链路不作为默认开箱承诺

## 1.2 判断结果输出格式
- 结论：更适合讲授 / 练习 / 互动
- 优先级：高 / 中 / 低
- 判断依据：
  - 学习目标：当前更偏理解概念、记忆知识点，还是应用做题
  - 学生阶段：学段是否匹配、基础是否足够、易错点是否集中
  - 课堂资源：课时是否够、材料是否齐、是否具备语音或题库支持
- 说明：用简短中文说明为什么这样安排
- 下一步建议：先讲、先练、先互动，或先补资料再设计

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认学科、学段、课时长度
2. 补齐教学与语音相关 Key
3. 演示课堂设计
4. 再演示语音能力

## 4. 什么时候需要先初始化 Key
- 只有在你要启用语音、互动课堂、题库运行或 LMS 联动时，才需要初始化对应凭证。
- 当前交付若仅做中文备课、讲义整理和课程说明，可先不配置这些外部 Key。

## 5. 风险与边界
- 不替代正式评分
- 不把语音误识别内容当事实
- 不替代老师做高风险学生判断
