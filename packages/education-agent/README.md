# 教育专业包

这是 `教育 Agent` 的正式交付包。

## 包定位
- 主要解决：备课、题目、讲义解析、语音课堂接入与老师讲解交付
- 交付对象：管理员、实施顾问、培训讲师、终端用户
- 交付目标：可安装、可演示、可教学、可验证

## 当前交付边界
- 当前可直接交付的是：中文教学设计、讲义整理、课程说明、基础课堂辅助问答。
- 语音转写、语音播报、课堂互动、LMS 对接等能力需要额外完成外部配置或第三方平台接入。
- 文档提到的 OpenMAIC、Quizzle、Moodle 等来源用于说明接入方向，不代表当前环境已默认打通。

## 判断型输出结构
- 结论
- 优先级
- 判断依据
  - 学习目标
  - 学生阶段
  - 课堂资源
- 说明
- 下一步建议

## 包内包含
- quickstart
- user-guide
- trainer-guide
- faq
- examples
- install-manifest
- image-plan

## 交付原则
- 中文优先
- 讲师可直接讲
- 管理员可直接安装
- 用户可直接上手
- 风险边界必须可见
