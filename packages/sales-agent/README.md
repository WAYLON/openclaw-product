# 客服销售专业包

这是 `客服 / 销售转化 Agent` 的正式交付包。

## 包定位
- 主要解决：FAQ、线索识别、跟进推进、预约转化与记录沉淀
- 交付对象：管理员、实施顾问、培训讲师、终端用户
- 交付目标：可安装、可演示、可教学、可验证

## 当前交付边界
- 当前可直接交付的是：中文客服问答口径、线索判断建议、会话总结与流程说明。
- 工单台、CRM、预约、签署、问卷回收等能力需要额外完成外部配置或第三方平台接入。
- 文档中提到的 Chatwoot、UVdesk、EspoCRM、Formbricks、Cal.com、Documenso 等名称，不等于当前默认开箱可用。

## 判断型输出结构
- 结论
- 转化概率
- 判断依据
  - 意向强度
  - 当前阶段
  - 下一步动作判断
- 说明
- 下一步建议

## 包内包含
- quickstart
- user-guide
- trainer-guide
- faq
- examples
- install-manifest
- image-plan

## 交付原则
- 中文优先
- 讲师可直接讲
- 管理员可直接安装
- 用户可直接上手
- 风险边界必须可见
