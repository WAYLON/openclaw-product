# 客服 / 销售转化 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，系统会先提示你配置缺失凭证。

## Q2：为什么不让它顺手做别的业务？
A：因为每个 Agent 都是平行独立关系，不设计业务总控 Agent。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：为什么回复里会同时给客户版和内部版？
A：因为这个 Agent 既要服务客户，也要方便团队沉淀记录。

## Q6：为什么遇到投诉不继续推单？
A：因为投诉和争议场景优先稳住服务与流程，而不是继续销售推进。
