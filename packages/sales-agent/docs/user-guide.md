# 客服 / 销售转化 Agent 用户指南

## 1. 用户能获得什么
- FAQ 响应
- 工单分流
- 线索判级
- 跟进记录
- 预约与合同确认引导

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认当前会话类型
2. 补齐客服/CRM/预约相关 Key
3. 演示回复建议
4. 演示跟进记录

## 4. 什么时候需要先初始化 Key
- chatwoot_api_key
- uvdesk_api_key
- espocrm_api_key
- formbricks_api_key
- calcom_api_key

## 5. 风险与边界
- 不能夸大承诺
- 不能违规外呼
- 不能把投诉场景硬推成交
