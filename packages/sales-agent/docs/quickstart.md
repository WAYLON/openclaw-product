# 客服 / 销售转化 Agent Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
客服 / 销售转化 Agent 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：FAQ、线索识别、跟进推进、预约转化与记录沉淀。

## 1.1 当前交付边界
- 当前可直接验证的是：中文回复建议、线索判断建议、流程说明。
- 工单台、CRM、预约、签署、问卷回收等能力，需要额外配置或第三方平台接入。
- 文档中的平台名称，仅表示可对接来源，不默认视为已接通。

## 1.2 判断结果结构
- 结论
- 转化概率
- 判断依据
  - 意向强度
  - 当前阶段
  - 下一步动作判断
- 说明
- 下一步建议

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 客服 / 销售转化 Agent 包。
3. 绑定一个渠道实例到当前 Agent。
4. 如需启用客服台、CRM 或预约等平台能力，再检查并初始化相关凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请判断这段咨询属于 FAQ、销售机会还是投诉。
- 请给一版客户回复和一版内部 CRM 跟进建议。
- 请判断客户所处销售阶段并给出下一步动作。

## 4. 首次讲解重点
1. 先讲服务和销售一体化
2. 再演示回复与跟进
3. 最后讲投诉边界
