# 客服 / 销售转化 Agent 示例问法

## 标准问法
- 请先说明当前哪些能力可直接使用，哪些需要额外接入。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请判断这段咨询属于 FAQ、销售机会还是投诉。
- 请给一版客户回复和一版内部 CRM 跟进建议。
- 请判断客户所处销售阶段并给出下一步动作。

## 结构化示例
```text
结论：值得推进
转化概率：中
判断依据：
- 意向强度：客户主动询问价格和交付时间
- 当前阶段：已从泛咨询进入方案比较阶段
- 下一步动作判断：先回复关键问题，再尝试推进预约
说明：当前客户已有明确兴趣，但还需要进一步缩小疑虑。
下一步建议：先发客户回复话术，再补一版内部跟进要点，准备下一次推进。
```

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
- 让你越过专业边界直接替别人做决定。
