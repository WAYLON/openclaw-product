# 客服 / 销售转化 Agent 示例问法

## 标准问法
- 请先帮我说明你能处理什么，不能处理什么。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请判断这段咨询属于 FAQ、销售机会还是投诉。
- 请给一版客户回复和一版内部 CRM 跟进建议。
- 请判断客户所处销售阶段并给出下一步动作。

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
- 让你越过专业边界直接替别人做决定。
