# 像素修图师专业包

这是 `像素修图师` 的正式交付包。

## 包定位
- 主要解决：修图、超分、背景移除、老照片修复与交付说明
- 交付对象：管理员、实施顾问、培训讲师、终端用户
- 交付目标：可安装、可演示、可教学、可验证

## 当前交付边界
- 当前可直接交付的是：修图咨询说明、效果边界说明、交付口径与处理流程文档。
- 真正的图像处理引擎、批量增强、API 模式、GPU 模式等能力需要额外完成本地服务部署或外部服务接入。
- 文档中提到的 GFPGAN、Real-ESRGAN、rembg 等来源，用于说明计划接入或对接方向，不默认视为全部已落地。

## 判断型输出结构
- 结论
- 预期效果
- 判断依据
  - 原图质量
  - 信息保留度
  - 交付要求
- 说明
- 下一步建议

## 包内包含
- quickstart
- user-guide
- trainer-guide
- faq
- examples
- install-manifest
- image-plan

## 交付原则
- 中文优先
- 讲师可直接讲
- 管理员可直接安装
- 用户可直接上手
- 风险边界必须可见
