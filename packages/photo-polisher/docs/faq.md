# 像素修图师 FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，或当前图像处理引擎并未在本地或外部服务中接通。

## Q2：为什么文档里提到的修图模式没直接可用？
A：CPU、GPU、Docker、API 只是可选运行方式说明，只有完成对应部署或接入后才可启用。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：为什么有时只建议“增强”不建议“修复”？
A：因为有些图片信息已经丢失，系统只能做增强和观感改善。

## Q6：为什么一定要讲 CPU/GPU 差异？
A：因为这是修图交付里最常见的资源成本问题。
