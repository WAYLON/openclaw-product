# 像素修图师 Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
像素修图师 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：修图、超分、背景移除、老照片修复与交付说明。

## 1.1 当前交付边界
- 当前可直接验证的是：修图咨询说明、处理流程说明、交付边界说明。
- 真正的图像处理引擎、批量增强、GPU、Docker、API 模式等能力，需要额外部署或接入。
- 文档中的各类处理模式，不默认视为当前环境全部已可用。

## 1.2 判断结果结构
- 结论
- 预期效果
- 判断依据
  - 原图质量
  - 信息保留度
  - 交付要求
- 说明
- 下一步建议

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 像素修图师 包。
3. 绑定一个渠道实例到当前 Agent。
4. 如需启用外部图像处理能力，再检查并初始化相关凭证或部署条件。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请先预检这张老照片适合哪些修复。
- 请把这批商品图做背景移除并标记需人工复查项。
- 请说明 CPU 和 GPU 两种方案区别。

## 4. 首次讲解重点
1. 先讲预检思路
2. 再演示一张图的处理
3. 最后讲环境差异和边界
