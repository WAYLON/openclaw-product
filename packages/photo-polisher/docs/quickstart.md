# 像素修图师 Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
像素修图师 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：修图、超分、背景移除、老照片修复与交付说明。

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 像素修图师 包。
3. 绑定一个渠道实例到当前 Agent。
4. 检查并初始化必要凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请先预检这张老照片适合哪些修复。
- 请把这批商品图做背景移除并标记需人工复查项。
- 请说明 CPU 和 GPU 两种方案区别。

## 4. 首次讲解重点
1. 先讲预检思路
2. 再演示一张图的处理
3. 最后讲环境差异和边界
