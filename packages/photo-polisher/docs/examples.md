# 像素修图师 示例问法

## 标准问法
- 请先说明当前哪些能力可直接使用，哪些需要额外部署或接入。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请先预检这张老照片适合哪些修复。
- 请把这批商品图做背景移除并标记需人工复查项。
- 请说明 CPU 和 GPU 两种方案区别。

## 结构化示例
```text
结论：只建议增强
预期效果：中
判断依据：
- 原图质量：分辨率偏低，噪点明显
- 信息保留度：主体轮廓还在，但细节已经丢失较多
- 交付要求：用户希望提升观感，不要求精细还原
说明：这张图适合做观感改善，不适合承诺高精度修复。
下一步建议：先做增强预览，若用户接受，再决定是否进入人工精修。
```

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
- 让你越过专业边界直接替别人做决定。
