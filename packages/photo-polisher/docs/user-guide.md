# 像素修图师 用户指南

## 1. 用户能获得什么
- 当前可直接使用：修图咨询说明、交付边界说明、处理模式解释
- 按接入情况可启用：图像预检、人脸修复、超分、背景移除、批量处理

## 1.1 当前交付状态说明
- 已内置：中文修图流程说明、效果边界说明、交付口径
- 需外部配置：如走外部 API 的图像处理能力
- 需第三方平台：无统一默认平台，需按具体引擎或服务单独接入
- 仅规划：CPU / GPU / Docker / API 多模式并不默认等于当前环境全部可用

## 1.2 判断结果输出格式
- 结论：值得修 / 只建议增强 / 建议人工处理
- 预期效果：高 / 中 / 低
- 判断依据：
  - 原图质量：分辨率是否过低、噪点是否明显、是否严重模糊或曝光异常
  - 信息保留度：五官或主体边缘是否还清楚、细节是否已经大面积丢失
  - 交付要求：用户要的是观感改善、精细修复，还是批量快速处理
- 说明：用简短中文说明能改善什么、不能承诺什么
- 下一步建议：直接增强、转人工精修、重新提供更清晰原图，或先做预检

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认图像问题类型
2. 确认本地或 API 模式
3. 演示预检与处理
4. 展示前后对比

## 4. 什么时候需要先初始化 Key
- 只有在你要启用外部图像服务时，才需要初始化对应凭证。
- 当前交付若仅做处理建议、模式说明和交付口径，可先不配置这些外部 Key。

## 5. 风险与边界
- 不承诺无限恢复
- 不伪造证据图片
- 不把猜测补全当真实修复
