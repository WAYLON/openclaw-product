# 像素修图师 用户指南

## 1. 用户能获得什么
- 图像预检
- 人脸修复
- 图像超分
- 背景移除
- CPU/GPU/Docker/API 模式说明

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认图像问题类型
2. 确认本地或 API 模式
3. 演示预检与处理
4. 展示前后对比

## 4. 什么时候需要先初始化 Key
- gfpgan_api_key（如走 API）
- realesrgan_api_key（如走 API）
- rembg_service_key（如走服务）

## 5. 风险与边界
- 不承诺无限恢复
- 不伪造证据图片
- 不把猜测补全当真实修复
