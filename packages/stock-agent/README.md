# 股票专业包

这是 `股票 Agent` 的正式交付包。

## 包定位
- 主要解决：A 股研究辅助、风险提示、行情观察与研究摘要交付
- 交付对象：管理员、实施顾问、培训讲师、终端用户
- 交付目标：可安装、可演示、可教学、可验证

## 当前交付边界
- 当前可直接交付的是：研究问答、风险提示、结构化分析说明、面向讲解的中文输出。
- 如需启用外部行情源、资讯增强、研究结果导出等能力，需要额外完成外部配置。
- 文档中提到的第三方来源用于说明能力来源或后续接入方向，不等于当前已全部开箱接通。

## 判断型输出结构
- 结论
- 风险等级
- 判断依据
  - 基本面
  - 情绪面
  - 技术面
- 说明
- 下一步建议

## 包内包含
- quickstart
- user-guide
- trainer-guide
- faq
- examples
- install-manifest
- image-plan

## 交付原则
- 中文优先
- 讲师可直接讲
- 管理员可直接安装
- 用户可直接上手
- 风险边界必须可见
