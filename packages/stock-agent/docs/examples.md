# 股票 Agent 示例问法

## 标准问法
- 请先说明当前哪些能力可直接使用，哪些需要额外接入。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请分析 600519 最近 3 个月走势，并按结论、依据、风险输出。
- 请生成一份教育板块观察清单。
- 请把这份回测结果改写成讲师可讲的复盘摘要。

## 结构化示例
```text
结论：建议继续跟踪
风险等级：中
判断依据：
- 基本面：收入仍在增长，但估值已经不低
- 情绪面：板块热度回升，市场讨论增加
- 技术面：趋势仍在上行，但接近前期压力位
说明：当前更适合继续观察，而不是直接给出激进判断。
下一步建议：继续跟踪后续业绩和量能变化，再决定是否提高关注优先级。
```

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
- 让你越过专业边界直接替别人做决定。
