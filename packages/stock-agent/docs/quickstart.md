# 股票 Agent Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
股票 Agent 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：A 股研究辅助、风险提示、行情观察与研究摘要交付。

## 1.1 当前交付边界
- 当前可直接验证的是：中文研究问答、风险提示、结构化研究输出。
- 行情源、资讯增强、同花顺智能选股增强、研究结果导出等能力，需要完成额外配置后再启用。
- 文档里提到的第三方来源，不默认等于当前环境已经接通。

## 1.2 判断结果结构
- 结论
- 风险等级
- 判断依据
  - 基本面
  - 情绪面
  - 技术面
- 说明
- 下一步建议

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 股票 Agent 包。
3. 绑定一个渠道实例到当前 Agent。
4. 如需启用外部能力，再检查并初始化相关凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请分析 600519 最近 3 个月走势，并按结论、依据、风险输出。
- 如果已接入同花顺智能选股，请按“低位放量 + 趋势转强”给我一份候选股票观察清单。
- 请生成一份教育板块观察清单。
- 请把这份回测结果改写成讲师可讲的复盘摘要。

## 4. 首次讲解重点
1. 先讲研究辅助定位
2. 再演示一只股票的完整分析链路
3. 最后强调风险边界与不能做的事
