# 股票 Agent Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
股票 Agent 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：A 股研究辅助、风险提示、行情观察与研究摘要交付。

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 股票 Agent 包。
3. 绑定一个渠道实例到当前 Agent。
4. 检查并初始化必要凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请分析 600519 最近 3 个月走势，并按结论、依据、风险输出。
- 请生成一份教育板块观察清单。
- 请把这份回测结果改写成讲师可讲的复盘摘要。

## 4. 首次讲解重点
1. 先讲研究辅助定位
2. 再演示一只股票的完整分析链路
3. 最后强调风险边界与不能做的事
