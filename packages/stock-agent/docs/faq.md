# 股票 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，系统会先提示你配置缺失凭证。

## Q2：为什么不让它顺手做别的业务？
A：因为每个 Agent 都是平行独立关系，不设计业务总控 Agent。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：为什么不直接告诉我买还是卖？
A：因为首发定位是研究辅助与风险提示，不是荐股或自动交易系统。

## Q6：为什么结论里总会带风险？
A：这是硬性要求，股票 Agent 必须把不确定性和反例一起讲清楚。
