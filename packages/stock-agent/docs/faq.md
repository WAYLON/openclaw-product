# 股票 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，或当前问题依赖外部能力尚未接入。系统会先提示你缺少什么，而不是假装已经具备。

## Q2：为什么文档里提到的能力没有直接可用？
A：因为有些能力只是对接来源或增强方向，需要额外配置行情、资讯或导出链路后才可启用。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：为什么不直接告诉我买还是卖？
A：因为首发定位是研究辅助与风险提示，不是荐股或自动交易系统。

## Q6：为什么结论里总会带风险？
A：这是硬性要求，股票 Agent 必须把不确定性和反例一起讲清楚。
