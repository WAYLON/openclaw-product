# 股票 Agent 讲师讲解手册

> 这是正式交付物，面向老师、实施顾问、售前顾问。

## 讲解目标
- 让客户理解这个 Agent 解决什么问题。
- 让客户知道首次怎么装、怎么配、怎么测。
- 让客户知道风险边界与最佳实践。

## 推荐讲解顺序
1. 先讲研究辅助定位
2. 再演示一只股票的完整分析链路
3. 最后强调风险边界与不能做的事

## 讲师话术模板
- 这不是一个总控机器人，而是一个独立专业 Agent。
- 它有自己的技能绑定、记忆与文档；工作区核心文件由官方生成。
- 它主要解决的是：A 股研究辅助、风险提示、行情观察与研究摘要交付。
- 如果第一次调用外部能力缺少 Key，它不会直接报错，而会先提示你如何初始化。
- 如果客户已经接入 qveris-official，可继续演示同花顺智能选股相关的条件筛选与候选股观察。

## 标准演示脚本
- 请分析 600519 最近 3 个月走势，并按结论、依据、风险输出。
- 如果已接入同花顺智能选股，请按“低位放量 + 趋势转强”筛一批值得继续观察的股票。
- 请生成一份教育板块观察清单。
- 请把这份回测结果改写成讲师可讲的复盘摘要。

## 演示时的标准输出结构
- 讲师演示时，建议固定展示以下结构：
  - 结论
  - 风险等级
  - 判断依据
    - 基本面
    - 情绪面
    - 技术面
  - 说明
  - 下一步建议
- 建议讲师提醒客户：这里不是只讲观点，而是会给出“是否继续跟踪”的判断、风险等级和后续动作。

## 首次演示推荐案例
- 输入：
  - 请判断 600519 现在是否值得继续跟踪，并给出风险等级、依据和下一步建议。
- 输出：
  - 输出一份结构化判断结果，包括结论、风险等级、基本面/情绪面/技术面依据、说明和下一步建议。
- 为什么这个案例最适合第一次演示：
  - 它最容易让客户看见，这个 Agent 不是只会分析，而是会帮助做跟踪取舍和风险判断。

## 承诺边界提醒
- 可默认演示的能力：
  - 输入股票代码或研究主题，输出“是否继续跟踪”的判断结果。
  - 输入已有观点或摘要，输出可直接发给客户或团队的中文研究说明。
- 需先说明“按接入情况启用”的能力：
  - 接入行情 API 后，可补充实时价格、K 线和行情变化。
  - 接入 qveris-official 后，可补充同花顺智能选股类筛选、候选股清单和入选理由。
  - 接入外部资讯或导出链路后，可补充资讯增强和研究导出。
- 当前不应默认承诺的能力：
  - 未接入行情或资讯服务时，不会返回实时市场数据。
  - 未接入 qveris-official 时，不会默认返回同花顺智能选股结果或条件筛选名单。
  - 不会执行任何自动交易、下单或保证收益判断。

## 讲解话术建议
- 讲师可直接这样说：
  - 这不是普通聊天，而是“结论 + 风险等级 + 判断依据 + 下一步建议”的研究判断输出，方便团队直接决定要不要继续跟踪。

## 1分钟演示版本
- 推荐提问：
  - 这只股票现在还值得继续跟踪吗？
- 预期输出结构：
  - 会直接输出结论、风险等级、基本面/情绪面/技术面依据，以及下一步建议。
- 讲解话术：
  - 这不是普通聊天，它不是只会说看法，而是会给出结构化判断和下一步动作。
  - 客户看到的不是“聊股票”，而是“要不要继续跟”的明确输出。

## 重点风险边界
- 首发不做自动交易
- 不输出保证收益表达
- 不能替代正式投顾结论
