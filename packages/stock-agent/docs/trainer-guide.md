# 股票 Agent 讲师讲解手册

> 这是正式交付物，面向老师、实施顾问、售前顾问。

## 讲解目标
- 让客户理解这个 Agent 解决什么问题。
- 让客户知道首次怎么装、怎么配、怎么测。
- 让客户知道风险边界与最佳实践。

## 推荐讲解顺序
1. 先讲研究辅助定位
2. 再演示一只股票的完整分析链路
3. 最后强调风险边界与不能做的事

## 讲师话术模板
- 这不是一个总控机器人，而是一个独立专业 Agent。
- 它有自己的灵魂、提示词、技能绑定、记忆与文档。
- 它主要解决的是：A 股研究辅助、风险提示、行情观察与研究摘要交付。
- 如果第一次调用外部能力缺少 Key，它不会直接报错，而会先提示你如何初始化。

## 标准演示脚本
- 请分析 600519 最近 3 个月走势，并按结论、依据、风险输出。
- 请生成一份教育板块观察清单。
- 请把这份回测结果改写成讲师可讲的复盘摘要。

## 重点风险边界
- 首发不做自动交易
- 不输出保证收益表达
- 不能替代正式投顾结论
