# 股票 Agent 用户指南

## 1. 用户能获得什么
- 当前可直接使用：中文研究问答、风险提示、结构化分析说明
- 按接入情况可启用：行情与 K 线观察、资讯增强、同花顺智能选股筛选、研究结果导出

## 1.1 当前交付状态说明
- 已内置：研究口径整理、中文结论输出、风险边界提示
- 需外部配置：部分行情源、研究增强能力、同花顺智能选股接入
- 仅规划：文档导出链路如未单独交付，不作为默认开箱承诺

## 1.2 判断结果输出格式
- 结论：是否建议继续跟踪
- 风险等级：高 / 中 / 低
- 判断依据：
  - 基本面：收入是否连续增长、利润率是否异常波动、估值是否明显高于同类
  - 情绪面：近期消息是持续升温还是退潮、讨论热度是否异常、板块是否情绪共振
  - 技术面：趋势是在上行、震荡还是走弱，成交量是否异常，是否接近关键支撑或压力位
- 说明：用 2 到 4 句话说明结论为何成立
- 下一步建议：继续跟踪、补资料复核，或先暂停判断

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认标的与时间范围
2. 补齐行情/研究能力凭证
3. 输出研究结果
4. 强调风险边界

## 4. 什么时候需要先初始化 Key
- 只有在你要启用外部研究增强、外部行情源或同花顺智能选股筛选时，才需要先初始化相关凭证。
- 当前交付若仅做中文研究问答与风险提示，可先不配置外部 Key。

## 5. 风险与边界
- 首发不做自动交易
- 不输出保证收益表达
- 不能替代正式投顾结论
