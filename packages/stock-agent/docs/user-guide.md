# 股票 Agent 用户指南

## 1. 用户能获得什么
- 股票代码解析
- 行情与 K 线观察
- 基本面/技术面分析
- 新闻与情绪影响整理
- 研究摘要与风险提示

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认标的与时间范围
2. 补齐行情/研究能力凭证
3. 输出研究结果
4. 强调风险边界

## 4. 什么时候需要先初始化 Key
- tradingagents_api_key（如接外部服务）
- akshare 相关环境（如有）
- 行情源接入凭证（按部署方式可选）

## 5. 风险与边界
- 首发不做自动交易
- 不输出保证收益表达
- 不能替代正式投顾结论
