# 法务 / 合同流程 Agent 用户指南

## 1. 用户能获得什么
- 合同上传解析
- OCR 与表格提取
- 条款风险提示
- 版本比对
- 签署状态跟踪

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认文档类型
2. 补齐签署/OCR/流程相关 Key
3. 演示解析与对比
4. 演示签署状态跟踪

## 4. 什么时候需要先初始化 Key
- documenso_api_key
- formbricks_api_key
- ocr_service_key
- n8n_api_key

## 5. 风险与边界
- 不替代律师正式意见
- 不能自动签署未经确认文件
- 不能把 OCR 抽取当法律定性
