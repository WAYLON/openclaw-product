# 法务 / 合同流程 Agent 用户指南

## 1. 用户能获得什么
- 当前可直接使用：中文合同流程说明、风险提示口径、比对思路说明
- 按接入情况可启用：OCR 提取、版本比对、签署状态跟踪、证据包导出

## 1.1 当前交付状态说明
- 已内置：流程问答、风险提醒、摘要结构说明
- 需外部配置：OCR、文档处理增强、部分流程联动
- 需第三方平台：审批表单、签署平台、自动化流转
- 仅规划：未单独交付的导出链路不作为默认开箱承诺

## 1.2 判断结果输出格式
- 结论：可继续推进 / 需重点复核 / 建议律师介入
- 风险等级：高 / 中 / 低
- 判断依据：
  - 条款风险：付款、违约、责任、期限等条款是否失衡或模糊
  - 合规边界：是否缺审批、授权、附件、签名或必要留痕
  - 介入级别：是普通流程解释，还是已经涉及重大责任与正式法律判断
- 说明：用简短中文说明当前风险点和边界
- 下一步建议：继续流程、补材料复核、转律师介入，或暂停签署

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认文档类型
2. 补齐签署/OCR/流程相关 Key
3. 演示解析与对比
4. 演示签署状态跟踪

## 4. 什么时候需要先初始化 Key
- 只有在你要启用 OCR、审批、签署跟踪或自动化流转时，才需要初始化对应凭证。
- 当前交付若仅做中文流程说明、风险提示和比对思路说明，可先不配置这些外部 Key。

## 5. 风险与边界
- 不替代律师正式意见
- 不能自动签署未经确认文件
- 不能把 OCR 抽取当法律定性
