# 法务 / 合同流程 Agent 讲师讲解手册

> 这是正式交付物，面向老师、实施顾问、售前顾问。

## 讲解目标
- 让客户理解这个 Agent 解决什么问题。
- 让客户知道首次怎么装、怎么配、怎么测。
- 让客户知道风险边界与最佳实践。

## 推荐讲解顺序
1. 先讲合同流程而不是法律替代
2. 再演示解析和比对
3. 最后讲签署状态与留痕

## 讲师话术模板
- 这不是一个总控机器人，而是一个独立专业 Agent。
- 它有自己的灵魂、提示词、技能绑定、记忆与文档。
- 它主要解决的是：合同解析、条款比对、审批流转、签署跟踪与证据打包。
- 如果第一次调用外部能力缺少 Key，它不会直接报错，而会先提示你如何初始化。

## 标准演示脚本
- 请解析这份合同并标出重点复核条款。
- 请比对两个版本合同。
- 请生成一份签署流程说明。

## 重点风险边界
- 不替代律师正式意见
- 不能自动签署未经确认文件
- 不能把 OCR 抽取当法律定性
