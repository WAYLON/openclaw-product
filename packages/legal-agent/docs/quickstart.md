# 法务 / 合同流程 Agent Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
法务 / 合同流程 Agent 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：合同解析、条款比对、审批流转、签署跟踪与证据打包。

## 1.1 当前交付边界
- 当前可直接验证的是：中文流程说明、风险提示口径、比对思路说明。
- OCR、审批、签署跟踪、证据包导出等能力，需要额外配置或第三方平台接入。
- 文档中的法务流程平台与 OCR 来源，不默认视为当前已打通。

## 1.2 判断结果结构
- 结论
- 风险等级
- 判断依据
  - 条款风险
  - 合规边界
  - 介入级别
- 说明
- 下一步建议

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 法务 / 合同流程 Agent 包。
3. 绑定一个渠道实例到当前 Agent。
4. 如需启用 OCR、审批或签署流程能力，再检查并初始化相关凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请解析这份合同并标出重点复核条款。
- 请比对两个版本合同。
- 请生成一份签署流程说明。

## 4. 首次讲解重点
1. 先讲合同流程而不是法律替代
2. 再演示解析和比对
3. 最后讲签署状态与留痕
