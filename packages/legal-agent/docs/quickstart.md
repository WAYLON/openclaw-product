# 法务 / 合同流程 Agent Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
法务 / 合同流程 Agent 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：合同解析、条款比对、审批流转、签署跟踪与证据打包。

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 法务 / 合同流程 Agent 包。
3. 绑定一个渠道实例到当前 Agent。
4. 检查并初始化必要凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请解析这份合同并标出重点复核条款。
- 请比对两个版本合同。
- 请生成一份签署流程说明。

## 4. 首次讲解重点
1. 先讲合同流程而不是法律替代
2. 再演示解析和比对
3. 最后讲签署状态与留痕
