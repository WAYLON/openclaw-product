# 法务 / 合同流程 Agent 示例问法

## 标准问法
- 请先说明当前哪些能力可直接使用，哪些需要额外接入。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请解析这份合同并标出重点复核条款。
- 请比对两个版本合同。
- 请生成一份签署流程说明。

## 结构化示例
```text
结论：需重点复核
风险等级：中
判断依据：
- 条款风险：违约责任明显偏重，付款条件表述不够清楚
- 合规边界：当前附件和授权说明还不完整
- 介入级别：已接近需要律师进一步复核的范围
说明：流程层面可以继续整理，但条款层面不建议直接放行。
下一步建议：先补齐附件和授权说明，再把关键条款提交律师复核。
```

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
- 让你越过专业边界直接替别人做决定。
