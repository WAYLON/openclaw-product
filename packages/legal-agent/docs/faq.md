# 法务 / 合同流程 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，或当前请求依赖 OCR、审批、签署等外部能力尚未接入。

## Q2：为什么文档里提到的签署或导出能力没直接可用？
A：这些能力依赖第三方平台或外部导出链路；未接通时，不作为默认开箱承诺。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：为什么不能直接下法律结论？
A：因为本系统定位是法务流程与合同协作助手，不是执业法律意见系统。

## Q6：为什么要单独跟踪签署状态？
A：因为合同流程问题往往不在内容本身，而在流转和状态留痕。
