# 法务 / 合同流程 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，系统会先提示你配置缺失凭证。

## Q2：为什么不让它顺手做别的业务？
A：因为每个 Agent 都是平行独立关系，不设计业务总控 Agent。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：为什么不能直接下法律结论？
A：因为本系统定位是法务流程与合同协作助手，不是执业法律意见系统。

## Q6：为什么要单独跟踪签署状态？
A：因为合同流程问题往往不在内容本身，而在流转和状态留痕。
