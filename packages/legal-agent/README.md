# 法务合同流程专业包

这是 `法务 / 合同流程 Agent` 的正式交付包。

## 包定位
- 主要解决：合同解析、条款比对、审批流转、签署跟踪与证据打包
- 交付对象：管理员、实施顾问、培训讲师、终端用户
- 交付目标：可安装、可演示、可教学、可验证

## 当前交付边界
- 当前可直接交付的是：中文合同流程说明、风险提示口径、比对思路与摘要结构。
- OCR 深度解析、审批流、签署跟踪、证据包导出等能力需要额外完成外部配置或第三方平台接入。
- 文档中提到的 Documenso、MinerU、PaddleOCR、Formbricks、n8n 等来源，默认仅表示接入方向或依赖来源，不代表全部已接通。

## 判断型输出结构
- 结论
- 风险等级
- 判断依据
  - 条款风险
  - 合规边界
  - 介入级别
- 说明
- 下一步建议
