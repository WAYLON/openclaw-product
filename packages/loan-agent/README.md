# 助贷专业包

这是 `助贷 Agent` 的正式交付包。

## 包定位
- 主要解决：线索采集、资格预检、材料引导、预约与合规流程交付
- 交付对象：管理员、实施顾问、培训讲师、终端用户
- 交付目标：可安装、可演示、可教学、可验证

## 当前交付边界
- 当前可直接交付的是：中文流程说明、初步问答、材料清单说明、合规提醒。
- 线索表单、CRM 同步、预约、签署、OCR 深度解析等能力需要额外完成外部配置或第三方平台接入。
- 文档里提到的 Formbricks、EspoCRM、Cal.com、Documenso、OCR 服务等，不默认等于已接好。

## 判断型输出结构
- 结论
- 合规风险
- 判断依据
  - 材料完整度
  - 基础条件
  - 流程约束
- 说明
- 下一步建议

## 包内包含
- quickstart
- user-guide
- trainer-guide
- faq
- examples
- install-manifest
- image-plan

## 交付原则
- 中文优先
- 讲师可直接讲
- 管理员可直接安装
- 用户可直接上手
- 风险边界必须可见
