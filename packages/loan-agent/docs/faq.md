# 助贷 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，或当前请求依赖表单、CRM、OCR、预约等外部能力尚未接入。

## Q2：为什么文档里提到的流程能力没直接跑起来？
A：部分流程能力依赖第三方平台或外部服务，当前未接入时只会给说明，不会假装已打通。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：为什么不能直接告诉我能不能批？
A：因为助贷 Agent 首发只做引导、初筛和材料说明，不做正式授信结论。

## Q6：为什么要提示隐私边界？
A：因为材料里常含身份证、流水、收入证明等敏感信息。
