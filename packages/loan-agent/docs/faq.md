# 助贷 Agent FAQ

## Q1：为什么没有直接执行？
A：可能命中了 Key 初始化守卫，系统会先提示你配置缺失凭证。

## Q2：为什么不让它顺手做别的业务？
A：因为每个 Agent 都是平行独立关系，不设计业务总控 Agent。

## Q3：为什么默认中文？
A：这是平台硬规则，中文优先，英文可选。

## Q4：为什么有些高风险能力不能直接开？
A：首发强调可安装、可演示、可教学，但高风险能力仍需单独风控。

## Q5：为什么不能直接告诉我能不能批？
A：因为助贷 Agent 首发只做引导、初筛和材料说明，不做正式授信结论。

## Q6：为什么要提示隐私边界？
A：因为材料里常含身份证、流水、收入证明等敏感信息。
