# 助贷 Agent 示例问法

## 标准问法
- 请先说明当前哪些能力可直接使用，哪些需要额外接入。
- 如果需要初始化 Key，请先告诉我缺什么、去哪里配。
- 请用中文分步骤说明。
- 请先做资格预检并列出还缺哪些材料。
- 请把这份收入证明解析成中文摘要。
- 请生成一份面向客户的合规材料清单说明。

## 结构化示例
```text
结论：建议先补材料再推进
合规风险：中
判断依据：
- 材料完整度：缺近 6 个月流水和收入证明
- 基础条件：职业和收入描述基本清楚，但负债情况还不完整
- 流程约束：当前还不适合直接进入预约或签署阶段
说明：先补齐关键材料，比直接推进更稳妥，也更符合合规流程。
下一步建议：先发材料清单给客户，补齐后再做下一轮预检。
```

## 错误问法
- 什么都不说，直接替我全部自动执行。
- 没配置任何系统凭证就要求你直接打通外部平台。
- 让你越过专业边界直接替别人做决定。
