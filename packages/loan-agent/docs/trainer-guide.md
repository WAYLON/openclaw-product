# 助贷 Agent 讲师讲解手册

> 这是正式交付物，面向老师、实施顾问、售前顾问。

## 讲解目标
- 让客户理解这个 Agent 解决什么问题。
- 让客户知道首次怎么装、怎么配、怎么测。
- 让客户知道风险边界与最佳实践。

## 推荐讲解顺序
1. 先讲流程定位
2. 再演示材料清单和预检
3. 最后讲合规边界

## 讲师话术模板
- 这不是一个总控机器人，而是一个独立专业 Agent。
- 它有自己的灵魂、提示词、技能绑定、记忆与文档。
- 它主要解决的是：线索采集、资格预检、材料引导、预约与合规流程交付。
- 如果第一次调用外部能力缺少 Key，它不会直接报错，而会先提示你如何初始化。

## 标准演示脚本
- 请先做资格预检并列出还缺哪些材料。
- 请把这份收入证明解析成中文摘要。
- 请生成一份面向客户的合规材料清单说明。

## 重点风险边界
- 不替代授信审批
- 不承诺放款结果
- 不建议规避合规要求
