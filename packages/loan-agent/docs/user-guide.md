# 助贷 Agent 用户指南

## 1. 用户能获得什么
- 当前可直接使用：中文流程问答、材料清单说明、合规提醒
- 按接入情况可启用：线索采集、资格预检、OCR 材料解析、预约与合同流转

## 1.1 当前交付状态说明
- 已内置：流程解释、材料说明、合规边界提示
- 需外部配置：OCR 解析、部分表单或流程增强能力
- 需第三方平台：CRM、预约、签署、表单等平台型能力
- 仅规划：未单独交付的自动流转能力不作为默认开箱承诺

## 1.2 判断结果输出格式
- 结论：建议继续推进 / 先补材料 / 暂不推进
- 合规风险：高 / 中 / 低
- 判断依据：
  - 材料完整度：身份证、收入、流水、经营材料是否齐全且前后一致
  - 基础条件：收入是否稳定、负债压力是否明显、职业或经营情况是否说得清
  - 流程约束：当前更适合初筛、补件还是预约，是否已到正式提交流程
- 说明：用简短中文说明当前卡点或推进理由
- 下一步建议：补材料、继续沟通、预约顾问，或先暂停推进

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认客户目标
2. 补齐 CRM / 表单 / OCR / 预约相关 Key
3. 演示材料解析
4. 演示预约或合同流转

## 4. 什么时候需要先初始化 Key
- 只有在你要启用表单、CRM、预约、签署或 OCR 解析时，才需要初始化对应凭证。
- 当前交付若仅做中文流程说明、材料指引和合规提醒，可先不配置这些外部 Key。

## 5. 风险与边界
- 不替代授信审批
- 不承诺放款结果
- 不建议规避合规要求
