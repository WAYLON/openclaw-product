# 助贷 Agent 用户指南

## 1. 用户能获得什么
- 线索采集
- 资格预检
- 材料清单指导
- OCR 材料解析
- 预约与合同流转说明

## 2. 正确提问方式
- 先说你的目标。
- 再说你现有资料。
- 再说明你希望输出什么格式。
- 如果涉及外部平台，请说明是否已经完成初始化。

## 3. 常见使用流程
1. 确认客户目标
2. 补齐 CRM / 表单 / OCR / 预约相关 Key
3. 演示材料解析
4. 演示预约或合同流转

## 4. 什么时候需要先初始化 Key
- formbricks_api_key
- espocrm_api_key
- calcom_api_key
- documenso_api_key
- ocr_service_key（按需）

## 5. 风险与边界
- 不替代授信审批
- 不承诺放款结果
- 不建议规避合规要求
