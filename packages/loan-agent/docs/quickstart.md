# 助贷 Agent Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
助贷 Agent 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：线索采集、资格预检、材料引导、预约与合规流程交付。

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 助贷 Agent 包。
3. 绑定一个渠道实例到当前 Agent。
4. 检查并初始化必要凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请先做资格预检并列出还缺哪些材料。
- 请把这份收入证明解析成中文摘要。
- 请生成一份面向客户的合规材料清单说明。

## 4. 首次讲解重点
1. 先讲流程定位
2. 再演示材料清单和预检
3. 最后讲合规边界
