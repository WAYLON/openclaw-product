# 助贷 Agent Quick Start

> 适合首次安装、首次演示、首次培训。

## 1. 这个包是什么
助贷 Agent 是一个平行独立 Agent 包，不依赖业务总控 Agent。
它主要解决的是：线索采集、资格预检、材料引导、预约与合规流程交付。

## 1.1 当前交付边界
- 当前可直接验证的是：中文流程说明、材料清单说明、合规提醒。
- 表单、CRM、预约、签署、OCR 深度解析等能力，需要额外配置或第三方平台接入。
- 文档里提到的外部平台，不默认视为当前已经接好。

## 1.2 判断结果结构
- 结论
- 合规风险
- 判断依据
  - 材料完整度
  - 基础条件
  - 流程约束
- 说明
- 下一步建议

## 2. 5 分钟启动步骤
1. 安装平台基础包。
2. 安装 助贷 Agent 包。
3. 绑定一个渠道实例到当前 Agent。
4. 如需启用表单、OCR 或流程平台能力，再检查并初始化相关凭证。
5. 发起第一轮标准测试。

## 3. 首次测试建议
- 请先做资格预检并列出还缺哪些材料。
- 请把这份收入证明解析成中文摘要。
- 请生成一份面向客户的合规材料清单说明。

## 4. 首次讲解重点
1. 先讲流程定位
2. 再演示材料清单和预检
3. 最后讲合规边界
