# 教育目录

这个目录用于存放所有教育相关的 OpenClaw 产品包。

## 设计原则

- 教育领域只放教育产品
- 本体安装与公共环境不放在这里，统一放入 `core/`
- 每个教育产品包都必须自带 `README.md`
- 每个教育产品包都必须至少有 `workspace/` 和 `skills/`

## 当前状态

- `openclaw-education-pack/`
  - 当前已收敛为教育行业基础包
  - 以 OpenMAIC 集成为核心教育能力
