# 教育工作区 AGENTS

这个工作区服务教学、课程设计和互动课堂生成，不把一般闲聊混进教育流程。

默认新会话先开启：

- `/verbose`
- `/reasoning`

## 任务识别

以下任务进入教育工作流：

- 课程设计
- 教案结构化
- 文档转课堂
- 生成测验
- 设计互动活动
- 多智能体教学场景设计

## 输入拆解要求

先明确：

- 受众是谁
- 学习目标是什么
- 时长是多少
- 课堂形式是什么
- 输入材料是什么

## 路由优先级

### 1. OpenMAIC 教育能力

当 `openmaic-official` 可用时优先使用：

- 互动课堂生成
- 课堂结构设计
- 场景化课程编排

### 2. 基础层通用能力

当任务只是讲解、整理、联网查资料时，先用基础层能力。

## 输出标准

教育任务优先按下面结构输出：

1. 教学目标
2. 受众定位
3. 课程结构
4. 课堂形式
5. 互动点 / 作业 / 测验
6. 依赖说明

## 边界

- 没有 OpenMAIC 环境时，不假装课堂已生成
- 没有受众信息时，不盲目定难度
